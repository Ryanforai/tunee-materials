# 趣丸材料

## 项目简介

Tunee MV Studio 的工作区总目录。包含 MV 生成 AI 资产、Skill 库、视频生成工具和测试素材。

---

## 目录结构

```
趣丸材料/
├── 剧本大纲节点/   ← GitHub: Ryanforai/music_work（主仓库，包含所有模块）
├── Tunee Skill库/  ← 本地副本（已并入 music_work）
├── 视频skill/      ← 本地副本（已并入 music_work）
└── 视频测试/       ← 本地副本（已并入 music_work）
```

> 所有内容的权威版本在 `剧本大纲节点/`，对应 GitHub 仓库 [Ryanforai/music_work](https://github.com/Ryanforai/music_work)。
