# 模块⑧ 相机 + 胶片材质（Camera + Film Stock）
> 核心问题：**用什么「皮肤」包裹这张图？**
> 相机决定画面的底层质感；胶片决定色调的「偏色个性」；颗粒决定画面是否有呼吸感。

## 🔬 画面质感修饰词

### 颗粒感（Film Grain）
```
heavy film grain               ← 明显颗粒，胶片老照片感
moderate film grain            ← 适中颗粒，平衡感
subtle film grain              ← 轻微颗粒，增加质感不抢眼
fine-grained texture           ← 细腻颗粒，高清中加质感
heavy visible grain in shadows ← 阴部颗粒明显，暗部更有质感
coarse silver halide texture   ← 粗银颗粒感，强烈胶片物理感
```

### 镜头光学效果
```
anamorphic lens flare              ← 横向椭圆形光晕，最经典电影感
blue horizontal anamorphic streak  ← 蓝色横向光条，Panavision经典
oval bokeh                         ← 椭圆形焦外，变形宽银幕标志
visible halation effect            ← 高光红色光晕（CineStill专属）
lens vignetting                    ← 四角暗角，聚焦中心
chromatic aberration               ← 色差，老镜头特有
subtle lens distortion             ← 轻微桶形畸变，真实感
soft edge diffusion                ← 边缘柔化，梦境/回忆感
```

### 特殊冲洗效果
```
pushed development +2 stops     ← 推冲两档，颗粒更粗，反差更高
cross-processed look            ← 交叉冲洗，偏色失真，实验感
bleach bypass / skip bleach     ← 漂白跳过，低饱和高对比，战争/压抑感
```

## 📐 画幅比例
```
2.76:1 (Ultra Panavision 70)  ← 极宽史诗，昆汀/史诗战争
2.39:1 / 2.40:1 (Scope)       ← 宽幅电影画幅，最电影感，首推
1.85:1 (Flat)                  ← 标准电影画幅，稍宽于16:9
1.43:1 (IMAX)                  ← IMAX震撼画幅
1.37:1 (Academy)               ← 经典学院比例，年代感
4:3                            ← 古典/复古感
1:1                            ← 正方形，Instagram感
4:5                            ← 竖版，社交媒体人像
9:16                           ← 竖屏全幅，短视频/竖版MV
```
