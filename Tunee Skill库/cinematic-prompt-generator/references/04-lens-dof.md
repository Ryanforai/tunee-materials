# 模块④ 焦段 + 虚实（Lens + Depth of Field）
> 核心问题：**镜头怎么「看」这个世界——空间感是拉开、压缩还是平视？**
> 焦段控制空间透视关系；景深控制主体与背景的清晰分配。

---

## 焦段情绪字典

| 焦段 | 镜头类型 | 空间效果 | 情绪语言 | 最佳用途 |
|------|---------|---------|---------|---------|
| **8–14mm** | 鱼眼 / 超广角 | 极端畸变，边缘变形 | 怪诞、压迫、监控视角、失控感 | 赛博朋克仰拍、主观恐惧视角 |
| **16–24mm** | 广角 | 拉开纵深，强烈透视 | 临场感、浸入感、宏大叙事 | 风景、环境人像、纪实街拍 |
| **35mm** | 小广角 | 轻微透视，贴近人眼 | 亲历感、纪录片真实 | 街头人文、环境带感的人像 |
| **50mm** | 标准 | 最接近人眼自然视角 | 平静、客观、中性、真实 | 日常叙事、对话场景 |
| **85mm** | 人像标准 | 轻微空间压缩，完美焦外 | 唯美、优雅、商业高级感 | 人像特写、时尚、美妆 |
| **135mm** | 中长焦 | 明显空间压缩 | 疏离感、偷窥感、孤独 | 街头抓拍、人群中的孤独感 |
| **200mm+** | 长焦 | 强烈空间压缩，背景贴近 | 极度隔离、异化、望远镜视角 | 王家卫式人群中的主角 |

---

## 虚实控制（景深描述）

**原则：主体和背景必须分开描述，不能只写一个**

```
Shot on [焦段]mm [镜头类型] lens.
[主体部位] is/are tack sharp / razor sharp.
The background [背景虚化状态].
```

### 主体清晰度词库
```
tack sharp              ← 极度锐利，轮廓清晰
razor sharp             ← 刀锋般清晰
pin-sharp               ← 精准对焦
critically sharp        ← 专业术语：精确对焦点
```

### 背景状态词库
```
dissolves into smooth creamy bokeh           ← 奶油焦外，最常用
melts into abstract color fields             ← 化为抽象色块
collapses into soft, unrecognizable blur     ← 完全无法辨认
holds slight soft focus                      ← 轻微柔焦，背景仍可辨认
is compressed into a flat, layered backdrop  ← 长焦压缩感
remains fully sharp with deep depth of field ← 前后皆清晰（广角/风景用）
```

### 前景虚化（增加层次感）
```
Blurred foreground [元素] in the extreme near-field
Out-of-focus [植物/人群/栏杆] in the foreground adding depth layers
```

---

## 完整写法示例

### 人像（85mm，浅景深）
```
Shot on 85mm f/1.4 portrait lens.
Her face and pearl necklace are tack sharp.
The background dissolves into smooth creamy bokeh,
completely unrecognizable. A blurred foreground
candle flame occupies the lower-left near-field,
adding depth layering.
```

### 街头（35mm，中等景深）
```
Shot on 35mm lens, f/2.8.
The subject is razor sharp.
The background retains slight detail but holds
soft focus, neon signs identifiable but not distracting.
```

### 史诗风景（24mm，深景深）
```
Shot on 24mm wide-angle lens, f/8.
The lone figure in the foreground is tack sharp.
The vast mountain range behind remains fully sharp,
deep depth of field preserving the entire environment.
```

### 长焦孤独（135mm）
```
Shot on 135mm telephoto lens.
The subject is critically sharp.
The crowd behind is compressed into a flat,
layered backdrop of blurred faces and shapes —
present but unreachable.
```

## ⚠️ 物理逻辑禁忌（常见矛盾写法）

| 错误写法 | 矛盾原因 | 正确替代 |
|---------|---------|---------|
| `wide angle + extremely shallow depth of field` | 广角天生深景深，这是物理矛盾 | 改用 85mm+ 实现浅景深 |
| `long shot + extreme bokeh background` | 远景需要深景深交代环境 | 远景配深景深，或改中近景 |
| `fisheye + background compression` | 鱼眼拉开空间，压缩是长焦特性 | 背景压缩改用 135mm+ |
| `close-up + wide angle` | 广角特写导致极端面部变形（除非刻意效果） | 特写改用 85mm |

## 镜头类型修饰词

```
anamorphic lens         ← 变形镜头，产生椭圆形焦外和横向光晕
spherical lens          ← 球形镜头，标准圆形焦外
vintage lens            ← 老镜头，有色差和轻微暗角
cinema prime lens       ← 电影定焦头，最高品质渲染
```
