# 模块② 景别（Shot Size）
> 核心问题：**你想让观众站多远看这个画面？**
> 景别控制的是观众与画面人物之间的「心理距离」，直接决定情绪参与程度。

---

## 景别情绪字典

| 景别 | 提示词关键词 | 人物画面占比 | 触发情绪 |
|------|------------|------------|---------| 
| **极远景** | `Extreme long shot (ELS)` | 人物 < 5% | 孤独、渺小、宿命感、史诗 |
| **远景** | `Long shot (LS)` | 人物约 10–20% | 环境压迫、无力感、漂泊 |
| **全景** | `Full shot (FS)` | 人物全身入镜，头脚略有空间 | 人物力量感、全身服装呈现 |
| **中全景** | `Medium full shot / cowboy shot` | 大腿中部以上 | 西部片经典、动作前的静止 |
| **中景** | `Medium shot (MS), waist up` | 腰部以上 | 日常叙事、对话、平稳推进 |
| **中近景** | `Medium close-up (MCU), chest up` | 胸部以上 | 情绪聚焦、亲密感、内心状态 |
| **近景** | `Close-up (CU), shoulders up` | 肩膀以上 | 强烈情绪、直视冲击 |
| **大特写** | `Big close-up (BCU)` | 仅脸部充满画面 | 极度情绪、压迫感、眼神对决 |
| **极端特写** | `Extreme close-up (ECU)` | 眼睛/嘴唇/手等局部细节 | 微表情震撼、悬念、仪式感 |
| **插入镜头** | `Insert shot` | 物件或身体局部特写 | 叙事强调、线索呈现、意象 |
| **双人景** | `Two shot` | 两人同框，腰部以上 | 对话张力、关系呈现 |
| **过肩景** | `Over-the-shoulder shot (OTS)` | 前景人物后脑/肩，焦点在对方 | 电影对话感、视角代入 |
| **群体全景** | `Ensemble wide shot` | 多人全身同框 | 集体力量、舞台感、场景交代 |

---

## 特殊运动镜头（MV/商业常用）

| 镜头类型 | 提示词 | 视觉效果 | 最适合 |
|---------|-------|---------|-------|
| **推镜** | `slow push-in to close-up` | 逐渐靠近，情绪升温 | 高潮段落、情绪爆发前 |
| **拉镜** | `slow pull-back from close-up` | 逐渐疏远，情绪冷却或扩展视野 | 结尾、孤独感揭示 |
| **跟拍** | `tracking shot following subject` | 跟随人物移动，带入感强 | MV行走段落、叙事流动 |
| **环绕** | `360-degree orbit around subject` | 主体静止，镜头绕行 | MV标志性时刻、力量感呈现 |
| **手持晃动** | `handheld shaky camera` | 不稳定感，纪实粗粝感 | 独立摇滚MV、现场感、情绪失控 |
| **斯坦尼康** | `steadicam smooth follow shot` | 流畅跟随，介于稳定与手持之间 | 长走廊追逐、剧情片跟拍 |
| **航拍俯冲** | `drone diving shot` | 从高空俯冲至人物 | MV开场、地点交代、史诗感 |

## 视角修饰词（与景别搭配使用）

```
low angle shot          ← 仰拍，人物显得强大、压迫
high angle shot         ← 俯拍，人物显得渺小、脆弱
eye level shot          ← 平视，平等客观
bird's eye view         ← 极度俯视，上帝视角
worm's eye view         ← 极度仰视，压迫/敬畏感
dutch angle / canted    ← 斜角，不安、失衡、心理扭曲
over the shoulder shot  ← 过肩，电影对话感
point of view (POV)     ← 第一人称视角，代入感极强
```

## 每种景别的正确与错误用法

### 极远景 / 远景
✅ **适合**：人与自然的渺小关系、宿命感、史诗开篇  
✅ **示例**：孤独旅者在冰川雪原中跋涉、末世废土中的幸存者  
❌ **禁忌**：想表达狂喜、愤怒等强烈个人情绪（脸看不清，情绪传达不到）  
❌ **禁忌**：配合极浅景深（远景需要深景深交代环境）

### 全景 / 中全景（Cowboy Shot）
✅ **适合**：展示全身服装、肢体语言、人物与空间关系、MV站姿段落  
✅ **示例**：舞台登场、时装大片、荒野对决前的静止  
❌ **禁忌**：想表现内心崩溃（太远，面部细节不够）

### 中景
✅ **适合**：日常叙事基底，交代谁在什么地方做什么  
❌ **禁忌**：史诗战场大场面（太近，失去宏大感）

### 中近景 / 近景
✅ **适合**：情绪聚焦、情感对话、MV主歌段  
✅ **示例**：歌手演唱特写、情绪表达  
❌ **禁忌**：没有前期铺垫直接进入（生硬突兀）

### 大特写 / 极端特写
✅ **适合**：情绪高潮时刻、微表情捕捉、极度悲伤/恐惧  
✅ **示例**：泪水落下、眼神燃烧、嘴唇微颤  
❌ **禁忌**：配合广角镜头（广角特写会产生极端面部畸变）

---

## 核心原则

> **特写是情绪高潮的武器，不是越特写越高级。**
>
> 成熟的导演思维：先用远景/中景建立环境和人物关系，
> 在情绪需要爆发的关键时刻才切入特写——这时候的特写才有力量。

---

## 快速决策树

```
我想让观众感受到...

孤独/渺小/史诗     →  Extreme long shot / Long shot
人物全身+环境关系  →  Full shot / Cowboy shot
日常/平稳/对话     →  Medium shot, waist up
情绪/内心状态      →  Medium close-up, chest up
强烈情绪冲击       →  Close-up / Big close-up
微表情/细节震撼    →  Extreme close-up
两人关系/张力      →  Two shot / Over-the-shoulder shot
MV 动感追随        →  Tracking shot / Steadicam follow
```
