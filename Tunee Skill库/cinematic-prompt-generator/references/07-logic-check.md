# 模块⑦ 空间逻辑自洽验证（Spatial Logic Check）
> 这是整套公式的**质量门控**。
> 在输出最终提示词之前，必须逐项核查三条逻辑线。
> 发现矛盾 → 立即返回对应模块修正 → 重新验证。

---

## 为什么需要这个模块

AI 生成图片时，会尝试同时满足所有提示词指令。
当指令之间存在物理矛盾时，AI 会产生「缝合怪」——
半张脸清晰半张脸模糊、环境既清晰又虚化、光源既强烈又均匀覆盖……

这些矛盾在人类眼睛的判断里是「说不清哪里不对但就是不对劲」的根源。

---

## 验证线一：景别 × 焦段 × 景深 三角验证

这三个参数必须物理自洽，因为它们共同决定「空间的呈现方式」。

### ✅ 合法组合

| 景别 | 焦段 | 景深 | 空间效果 |
|------|------|------|---------|
| 极远景 / 远景 | 24–35mm 广角 | 深景深 | 环境叙事，前后皆清晰，人物渺小 |
| 全身中景 | 50mm 标准 | 中等景深 | 人物与环境并重，平稳叙事 |
| 中景 / 中近景 | 85mm 人像 | 浅景深 | 人像美化，背景化为柔美焦外 |
| 任何景别 | 135–200mm 长焦 | 浅景深 | 空间压缩，偷窥感，人群中孤立 |
| 特写 | 85–135mm | 极浅景深 | 面部细节极致放大，背景消失 |

### ❌ 矛盾组合 → 必须修正

| 矛盾写法 | 物理原因 | 修正方向 |
|---------|---------|---------|
| 远景 + 极浅景深 + 背景虚化 | 远景需要深景深才能交代环境，浅景深会让环境信息消失，违背远景初衷 | 改深景深，或改近景别 |
| 广角(16–24mm) + 背景压缩 | 广角拉开纵深使背景变小，背景压缩是长焦特性，两者相反 | 改用 135mm+ 实现压缩 |
| 鱼眼 + 背景压缩 | 鱼眼极端拉开空间，与压缩完全相反 | 不能共存 |
| 全景 + 浅景深 + 背景虚化 | 全景的叙事价值在于展示完整环境，虚化背景则摧毁这个价值 | 改深景深或改景别 |
| 特写 + 广角镜头（非刻意效果） | 广角特写导致严重面部变形，鼻子极大 | 改用 85mm 以上 |

## 验证线二：光影逻辑验证

### ✅ 合法组合

```
单一光源 + 大面积暗部 + 有明确衰减路径
  → 黑色电影感，高级戏剧感，物理真实

多光源 + 明确主次关系（主光源 vs 辅光源）
  → 商业感，多层次，自然感

窗光入射 + 对应方向的高亮区域 + 背对窗的暗部
  → 自然室内逻辑，真实感强
```

### ❌ 矛盾组合

```
practical lights only + 全局均匀明亮照明
  → practical lights 是局部点光源，不可能均匀覆盖全场
  → 修正：声明哪里亮哪里暗

暗部全黑，背景完全不可见 + 背景元素清晰可辨
  → 背景黑暗和背景清晰互相矛盾
  → 修正：二选一

强侧光 + 面部两侧同等亮度
  → 强侧光必然产生明显的亮暗面对比
  → 修正：声明亮面和阴影面的分布

overhead spotlight + 均匀照亮整个身体
  → 顶光只照亮顶部，迅速向下衰减
  → 修正：声明顶部亮，腰部以下消失入暗
```

## 验证线三：色彩逻辑验证

### ✅ 合法组合

```
主色调统治 >80% + 辅色 <20%，且辅色位置明确
  → 色调统一，画面干净有气质

同色系冷暖变化（如：深琥珀橙主导，浅金高光作变化）
  → 丰富但不乱

互补色主辅关系（如：暖橙主导，冷蓝点缀）
  → 经典电影色彩逻辑，张力感
```

### ❌ 矛盾组合

```
warm light + cool light 各占 50%
  → 无主次，两色互相抵消，画面灰脏
  → 修正：声明哪个是主色，哪个是 <20% 的点缀

neon colors + muted desaturated palette
  → 霓虹色是高饱和，muted是低饱和，根本矛盾
  → 修正：二选一

vibrant rainbow colors + cinematic color grading
  → 彩虹色调与电影统一色调本质矛盾
  → 修正：选定一个主色体系

色温混乱：tungsten warm light + daylight cool fill + neon green + red practical
  → 四种色温同时存在，没有主次，画面「脏」
  → 修正：选定主色温，其他作为小比例点缀
```

## 验证输出格式

每次生成提示词后，必须在输出中附上验证结果：

```
【⑦ 空间逻辑验证】

景别×焦段×景深：
  ✅ [中近景 + 85mm + 浅景深] — 物理自洽，人像标准逻辑

光影逻辑：
  ✅ [单一顶光 + 快速衰减 + 幕布反射填充] — 三层完整，方向自洽

色彩逻辑：
  ✅ [古金色主导90% + 深红点缀<10%] — 主次清晰，无色调冲突
```

如有矛盾：
```
【⑦ 空间逻辑验证】

景别×焦段×景深：
  ❌ 矛盾：远景 + 极浅景深 — 远景需要深景深
  → 修正：改为深景深 f/8，背景保持清晰
```

---

## 快速自查问题（输出前心算）

1. 我的景别和焦段是否同向？（都偏近 or 都偏远）
2. 光源在哪里？它能照亮我描述的区域吗？逻辑上通吗？
3. 我的主色和辅色有没有明确的比例声明？
4. 有没有两个互相矛盾的描述同时存在？
