# 模块⑥ 冷暖阶级（Color Hierarchy）
> 核心问题：**画面里谁是老大，谁是配角？**
> 色彩没有主次是画面「脏」的最大原因。一张图只能有一个主色统治者。

---

## 核心原则：80/20 法则

```
主色调  →  统治 80–90% 的画面面积
辅色调  →  只出现在 10–20% 的角落、缝隙、边缘
```

❌ 错误：`warm amber and teal color grading`（两色平等，各占50%，互相打架）  
✅ 正确：主色统治 + 辅色克制，明确声明比例

---

## 标准写法模板

```
The entire scene is dominated by [主色调描述].
Only [极小区域：the far background shadows / 
  the edge of the frame / a sliver of reflected light /
  the gap between curtains] catches a subtle, faint 
[辅色调], occupying no more than [10–20%] of the frame.
```

## 进阶技巧：色彩的「出现位置」也要声明

辅色出现的位置决定它与主色的「对话关系」：

**辅色在背景深处**（最常用，最自然）
```
Only the far background catches a faint [辅色] tint
```

**辅色在阴影缝隙**（神秘感）
```
Only the deepest shadow gaps reveal a subtle [辅色] undertone
```

**辅色在光源边缘**（光晕/色散效果）
```
Only the edges of the [光源] bleed a faint [辅色] halo
```

**辅色在反射面**（真实感）
```
Only the [镜面/水面/金属] catches a reflected hint of [辅色]
```

## 色调描述词库

### 暖色系（主色用）
```
deep warm amber                    ← 深琥珀橙
tungsten gold / tungsten orange    ← 钨丝灯橙（最电影感）
candlelight ivory                  ← 烛光象牙白
honey gold                         ← 蜂蜜金
burnt sienna                       ← 焦赭橙棕
matte antique gold                 ← 哑光古金
```

### 冷色系（主色或辅色）
```
cold cobalt blue                   ← 钴蓝
desaturated slate blue             ← 低饱和石板蓝
deep teal                          ← 深青蓝
cool silver-grey                   ← 冷银灰
icy blue-white                     ← 冰蓝白
```

### 辅色修饰词（让辅色显得「克制」）
```
subtle, faint [色调]               ← 轻微、隐约
barely perceptible [色调] tint     ← 几乎察觉不到
ghostly [色调] undertone           ← 幽灵般的底色
whisper of [色调]                  ← 一丝
```

## 完整示例

### 示例A：剧院暖金调
```
The entire scene is dominated by matte antique gold 
and warm candlelight tones, saturating every surface 
from the curtain fabric to the stage floor.
Only the narrow black gap between the curtains 
at the far back catches a barely perceptible 
deep crimson undertone, occupying less than 8% of the frame.
```

### 示例B：都市冷蓝调
```
The entire scene is dominated by cool desaturated 
silver-blue tones — the wet pavement, the fog, 
the building facades all unified in cold grey.
Only the subject's warm skintone and a single 
distant amber street lamp introduce warmth, 
together occupying no more than 15% of the frame.
```

### 示例C：好莱坞后台暖调
```
The entire scene is dominated by deep warm amber 
and tungsten gold, wrapping every surface in 
rich, decadent warmth.
Only the far mirror reflection and the deepest 
background shadows catch a faint, cold teal-blue 
presence — no more than 12% of the total frame area.
```

---

## ⚠️ 高频错误

```
❌ warm and cool tones             → 没有主次，画面脏
❌ rich colors                     → 无效描述，AI随机发挥
❌ vibrant color palette           → 多色竞争，无统治色
❌ dramatic color grading          → 情绪词代替物理色彩描述
✅ 始终声明：主色 + 辅色 + 辅色比例 + 辅色出现位置
```
