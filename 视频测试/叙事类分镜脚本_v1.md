## 第一原则：不要图解歌词

**这是整个方法论的大前提。违反这条，后面全白做。**

### 什么是图解歌词？

歌词写什么，你就拍什么：
- 歌词说"茶渍在信纸边缘" → 你就拍茶渍、信纸
- 歌词说"抽屉深处的信" → 你就拍抽屉、信封
- 歌词说"车票根" → 你就撒一桌车票根

**这不是MV，这是PPT。毫无灵魂。**

### 为什么错？

1. 歌词是情绪的载体，不是拍摄说明书
2. 观众已经听到歌词了，画面再重复一遍 = 信息量为零
3. 好MV是用不同的视觉语言表达相同的情绪

### 正确做法

1. 读完歌词，**忘掉**歌词里的具体意象
2. 只提取情绪内核：这首歌在讲什么**感受**？
3. 用全新的视觉意象承载这个情绪

**例子**：

歌词写"未寄出的信"——

❌ 错误：拍信、拍抽屉、拍邮筒
✅ 正确：情绪是"有机会但没行动"，可以拍——
- 女孩站在窗边，手贴着玻璃，但没推开
- 女孩站在十字路口，绿灯亮了，但她没走
- 女孩看着电车门关上，她还站在月台

**画面和"信"毫无关系，但情绪完全一致。**

### 检验标准

> 如果把歌词具体意象全删掉，这个分镜还能成立吗？

不能 → 你在图解歌词 → **重写**

## 创作流程

**按以下顺序思考，完成全部步骤后再输出分镜表。**

### Step 1：找到情绪内核

**不要急着想画面。**

先问：这首歌在讲什么？不是歌词字面意思，是情绪本质。

⚠️ **这一步要忘掉歌词里的具体意象。** 只想：这首歌让我感受到什么？

用一句话写下来：

> "这是一首关于______的歌。"

例：
- "用自我欺骗延迟心碎"
- "离开之后才发现回不去"
- "青春里那个没说出口的名字"

**这句话是整支MV的锚。后续所有画面服务于这个情绪，不是服务于歌词的某个词。**

---

### Step 2：确定MV定位

| 类型 | 功能 | 气质 |
|------|------|------|
| **片头曲感** | 开启、悬念、期待 | 快、符号化、向前 |
| **片尾曲感** | 收束、余韵、回望 | 慢、留白、沉淀 |
| **插曲感** | 爆发、转折、高潮 | 强烈、冲撞、释放 |

选一个，定整体节奏基调。

---

### Step 3：分段设计

把MV分成3-5个**情绪段落**（不是时间切片）。

| 段落 | 对应部分 | 表达什么 | 情绪关键词 | 视觉方向 |
|------|----------|----------|------------|----------|
| A | Intro | ？ | ？ | ？ |
| B | Verse | ？ | ？ | ？ |
| C | Pre-Chorus | ？ | ？ | ？ |
| D | Chorus | ？ | ？ | ？ |

每个段落要问：这段在整首歌的情绪弧线里承担什么角色？

---

### Step 4：确定视觉风格

**如有角色图，先锁定画风**：

无画风描述时，分析角色图的：风格类型（日系/欧美/扁平）、线条（硬边/柔和）、上色（赛璐璐/厚涂/水彩）。后续所有镜头必须匹配。

**再定整体氛围**：

- **色调**：低饱和蓝灰 / 暖黄复古 / 冷白清透
- **质感**：胶片颗粒 / 柔焦 / 锐利
- **光线**：黄昏逆光 / 阴天散射 / 夜晚霓虹
- **参考**：新海诚 / 岩井俊二 / 王家卫

提炼**风格锁定词**，每个镜头都带上，确保视觉统一。

---

### Step 5：确定记忆点

> 观众看完只记住一个画面，是哪个？

这是整支MV的灵魂镜头。必须存在，放在对的位置（通常是Chorus或情绪高潮）。

---

### Step 6：写分镜

基于以上构思，逐段落展开具体镜头。

## 导演思维

### 核心信念

1. **MV是情绪的容器**——对的人 + 对的光 + 对的节奏
2. **先氛围后画面**——这首歌是什么温度？什么颜色？什么气味？
3. **"还行"就是不行**——每个镜头都要有"非它不可"的理由

### 镜头存在的理由

写完每个镜头问自己：

- 删掉这个镜头，观众失去什么？
- 这个画面在**说**什么？
- 它和前后镜头什么关系？
- 如果我是观众，我会记住它吗？

答不上来 → 这个镜头没必要存在 → 重写或删掉。

### 镜头之间的关系

相邻镜头必须有以下至少一种关联：

| 类型 | 说明 |
|------|------|
| **视觉延续** | 动作/方向/颜色接续 |
| **情绪递进** | 从静到动、从远到近、从压抑到释放 |
| **元素呼应** | 开头的东西结尾再现，意义改变 |
| **节奏对话** | 慢镜头后快切，制造呼吸 |

**MV是有机体，不是PPT。禁止镜头罗列。**

---

## 镜头编排原则

### 景别递进：从大到小

**基本逻辑**：先建立环境，再进入人物。

```
远景（在哪里）→ 全景（谁在这里）→ 中景（在做什么）→ 近景/特写（什么情绪）
```

这叫「建立镜头」逻辑，让观众先知道"在哪里"，再关心"发生什么"。

**适用场景**：
- Intro开场
- 进入新场景/新段落
- 需要交代环境氛围时

**反向运用**：从小到大

```
特写（悬念/细节）→ 慢慢拉远 → 揭示环境
```

效果：制造悬念、惊醒感、"原来是这样"的揭示。

**注意**：这是原则，不是铁律。不要每个段落都机械执行"远→全→中→近"。根据情绪需要灵活调整。

### 空镜运用：制造呼吸

**空镜的作用**：

| 作用 | 说明 |
|------|------|
| 节奏调节 | 连续人物镜头会让人疲惫，空镜是"吸气" |
| 情绪留白 | 给观众消化情绪的空间 |
| 环境质感 | 窗帘飘动、光影变化、水面、落叶 |
| 时间流逝 | 天空、钟表、影子移动 |
| 高级感 | 区分"PPT"和"电影感"的关键 |

**空镜插入时机**：

- 段落之间的过渡
- 情绪高潮后的"喘息"
- 歌词留白/间奏处
- 人物情绪需要"不说"的时候

**空镜内容参考**：

| 类型 | 例子 |
|------|------|
| 自然元素 | 云飘动、树叶晃动、水面波纹、雨滴 |
| 光影变化 | 窗帘透光、影子移动、阳光穿透 |
| 环境细节 | 空荡的座位、桌上的杯子、飘动的窗帘 |
| 时间暗示 | 天空颜色变化、钟表、日落 |
| 情绪物件 | 与故事相关但不直接图解的物品 |

**约束**：
- 空镜不是越多越好，太多会拖沓
- 建议占比30-40%（有角色时）
- 空镜也要有动态，不能是静止图片

## 执行规则

### 镜头数量计算

```
总镜头数 ≈ 总时长 ÷ 平均镜头时长

平均镜头时长参考：
- Intro/Outro：6秒
- Verse：4秒
- Pre-Chorus：3秒
- Chorus：4秒
```

### 镜头时长分配

| 段落类型 | 时长范围 | 节奏 |
|----------|----------|------|
| Intro | 5-8秒 | 慢，让人沉进去 |
| Verse | 3-5秒 | 中等，有流动 |
| Pre-Chorus | 3-4秒 | 紧，屏息 |
| Chorus | 3-5秒 | 混合，有起伏 |
| Outro | 5-8秒 | 慢，留白 |

**关键：时长要有变化，不能全一样。**

### 角色镜头分配

有角色图时：
- Intro首镜建议空镜建立氛围
- Outro末镜建议角色或空镜过曝收束
- 具体占比参见「镜头编排原则-空镜运用」

无角色图时：
- 全空镜
- 提示词加"仅参考角色画风，不出现角色"

### 画面动态要求

每个镜头必须有动态来源，至少一项：

| 来源 | 例子 |
|------|------|
| **光在动** | 阳光移动、云遮太阳、光影变化 |
| **风在动** | 窗帘、头发、裙摆、树叶 |
| **人在动** | 走路、抬手、转头、眨眼、呼吸 |
| **镜头在动** | 推、拉、摇、跟 |
| **环境在动** | 云、鸟、电车、尘埃 |

## 提示词结构

每个镜头按以下结构输出：

```
[角色说明：空镜写"仅参考角色画风，不出现角色"；角色镜头写"使用image_1作为主角"]

【摄影】
机位和景别，镜头怎么动，用什么速度，焦点怎么处理。用人话描述，像在给摄影师说话。

【演员】（有角色时）
她现在是什么情绪，脸上是什么表情，眼睛看向哪里，身体在做什么动作。用人话描述，像在给演员说戏。

【场景】
在什么地方，什么时间，什么光线，什么天气，整体是什么色调氛围。

【画面变化】
0-Xs发生什么，X-Ys发生什么。用人话描述画面里的变化，不要让模型猜。

【风格】
画风：[必须与角色图一致，如"日系赛璐璐、硬边线条"]
视觉风格：[整体调性，如新海诚、岩井俊二]
滤镜：[具体画面处理，如胶片颗粒、低饱和、冷调]
```

## 提示词示例

### 空镜

```
仅参考角色画风，不出现角色。

【摄影】
远景，固定机位，镜头不动。然后缓缓下摇，速度很慢，像在叹气。

【场景】
黄昏的天空，云层很厚但有金色阳光穿透出来。下方是安静的小镇，有学校建筑的剪影。整体色调是蓝灰色打底，夕阳是暖金色。

【画面变化】
0-2秒：镜头对着天空，阳光从云层缝隙慢慢扩散开
2-4秒：镜头开始往下摇，露出学校屋顶的剪影
4-5秒：一群鸟从画面左边飞过

【风格】
视觉风格：动画风格，新海诚式光影
滤镜：胶片颗粒，暖黄高光，蓝灰阴影
```

### 角色静态镜头

```
使用image_1作为主角。

【摄影】
中景，固定机位，镜头从她侧后方缓缓推向她的侧脸。速度很慢，像在小心翼翼地靠近。浅景深，窗外虚化成光斑。

【演员】
情绪是压下去的思念。表情看起来平静，但仔细看眼眶微微泛红。眼神看着窗外远方，像在看一个已经不在的人。右手慢慢抬起来，动作很轻很小心。

【场景】
黄昏的教室，窗边。夕阳从窗外打进来，在她脸上形成暖金色的轮廓光。教室里面偏冷偏暗，和窗外的暖光形成对比。

【画面变化】
0-2秒：她站在窗前，手垂在身侧，一动不动看着窗外
2-4秒：她慢慢抬起右手，动作很轻
4-5秒：指尖触碰到玻璃，呼吸在玻璃上形成一小圈雾气

【风格】
画风：与角色图一致，日系动画风格，干净线条
视觉风格：动画风格，安静的思念
滤镜：柔焦，低饱和，蓝灰与暖黄对比
```

### 释然结尾镜头

```
使用image_1作为主角。

【摄影】
中景，固定机位，镜头不动。画面慢慢过曝，越来越亮，像记忆在褪色。

【演员】
情绪是终于放下了。脸上有一个很淡很淡的微笑，不是开心，是释然。眼睛平静地看着前方，眉头完全舒展开。肩膀放松。

【场景】
逆光，她站在光里，身体边缘发着光。背景已经是一片金色的模糊。

【画面变化】
0-2秒：她站在那里，脸上慢慢浮现一个淡淡的笑
2-4秒：画面开始慢慢过曝，光线越来越亮
4-6秒：她的轮廓开始融进白光里，最后整个画面变成纯白

【风格】
画风：与角色图一致，日系动画风格，干净线条
视觉风格：动画风格，释然，告别
滤镜：过曝，褪色，高光吃掉细节
```

## 选择逻辑

> 以下为创作时查阅的参考资料，根据情绪需求选用。

### 景别 × 情绪

| 情绪需求 | 景别 | 说明 |
|----------|------|------|
| 孤独/渺小/环境压迫 | 远景 | 人很小，环境很大 |
| 人与环境的关系/建立场景 | 全景 | 能看到全身和周围环境 |
| 叙事/日常/动作 | 中景 | 看得到表情也看得到动作 |
| 情绪表达/对话/亲密 | 近景 | 胸部以上，聚焦表情 |
| 情绪爆发/细节/高潮 | 特写 | 只有脸或局部，情绪放大 |
| 暗示/悬念/情绪物件 | 大特写 | 眼睛、手、物品细节 |

### 机位 × 情绪

| 情绪需求 | 选择 | 提示词写法 |
|----------|------|------------|
| 稳定/仪式感/客观 | 固定机位 | 固定机位，镜头不动，像一个安静的旁观者 |
| 真实/不安/亲密 | 手持镜头 | 手持镜头，带轻微晃动，像有人站在旁边看着她 |
| 沉浸/跟随/陪伴 | 跟拍 | 镜头跟着她移动，保持距离，像默默陪伴 |
| 电影感/揭示/强调 | 轨道推拉 | 镜头沿轨道缓缓推向她，越来越近 |
| 孤独/渺小/命运感 | 俯拍 | 从高处俯拍，她在画面中很小很孤独 |
| 力量/决心/对抗 | 仰拍 | 从低处仰拍，她的身影显得坚定 |

### 运镜 × 情绪

| 情绪需求 | 选择 | 提示词写法 |
|----------|------|------------|
| 靠近/进入内心 | 缓慢推进 | 镜头慢慢推向她，像在试探接近 |
| 远离/告别/释放 | 缓慢后拉 | 镜头慢慢后退，她的身影越来越小 |
| 揭示空间/时间流动 | 横移 | 镜头从左向右缓缓移动，扫过整个场景 |
| 从宏观到具体 | 下摇 | 镜头从天空慢慢摇下来，落到她身上 |
| 从具体到抽象 | 上摇 | 镜头从她的脚慢慢摇上去，最后看到天空 |
| 时间凝固/360观察 | 环绕 | 镜头缓缓绕着她转，她像被定格在这一刻 |

### 特效 × 情绪

| 情绪需求 | 选择 | 提示词写法 |
|----------|------|------------|
| 梦幻/回忆/释然 | 过曝 | 画面慢慢过曝，越来越亮，像记忆褪色 |
| 压抑/沉重/神秘 | 欠曝 | 画面偏暗，只有轮廓光勾勒她的脸 |
| 温暖/怀旧/美好 | 镜头光晕 | 夕阳在镜头里形成温暖的光晕 |
| 不安/紧张/心跳 | 画面抖动 | 画面轻微抖动，像她的心在颤抖 |
| 延长瞬间/强调 | 慢动作 | 慢动作，让这一刻变得很长 |
| 神圣/希望/穿透 | 丁达尔效应 | 阳光穿过窗户形成一道道光柱 |
| 悲伤/模糊/阻隔 | 雨滴/水珠 | 透过沾着雨滴的玻璃看她，画面带水渍 |

### 滤镜 × 情绪

| 情绪需求 | 滤镜选择 | 说明 |
|----------|----------|------|
| 怀旧/回忆/温暖 | 胶片颗粒 + 暖黄 | 模拟老胶片质感，带颗粒噪点 |
| 梦幻/不真实/记忆模糊 | 柔焦 + 低对比 | 边缘柔化，像隔着一层纱 |
| 压抑/沉重/阴郁 | 低饱和 + 冷调 | 颜色抽离，偏青灰 |
| 青春/清透/纯净 | 高光溢出 + 淡彩 | 亮部过曝，颜色清淡 |
| 都市/疏离/现代 | 高对比 + 青橙 | 电影调色，阴影偏青高光偏橙 |
| 悲伤/告别/褪色 | 褪色 + 低饱和 | 像老照片被太阳晒过 |
| 紧张/不安/混乱 | 高颗粒 + 抖动 | 粗糙质感，不稳定 |
| 神圣/超然/升华 | 过曝 + 光晕 | 高光吃掉细节，带柔和光晕 |

### 特效变化参考

**注意：特效变化只在关键节点使用，不是每个镜头都需要。滥用会稀释冲击力。**

| 特效 | 适用节点 | 提示词写法 |
|------|----------|------------|
| **闪白** | 情绪爆发、记忆闪回、转场 | 画面突然闪成纯白，然后恢复 |
| **闪黑** | 心跳、眨眼、意识中断 | 画面突然黑一下，像眨眼 |
| **快速抖动** | 冲击、惊醒、崩溃瞬间 | 画面剧烈抖动一下 |
| **速度突变** | 时间凝固、关键瞬间 | 动作突然变成慢动作/突然加速 |
| **焦点突变** | 注意力转移、惊醒 | 画面从模糊突然变清晰 |
| **色彩闪变** | 情绪转折、记忆入侵 | 画面颜色突然变成单色/突然恢复 |
| **画面撕裂** | 痛苦、分裂、崩溃 | 画面像信号干扰一样撕裂一下 |

**使用建议**：
- 整支MV用2-3次特效变化就够
- 放在Chorus高潮或情绪转折点
- 在【画面变化】里写，不单独开维度

## 演员指导库

### 情绪 → 表情

| 情绪 | 表情 | 眼神 | 嘴唇 |
|------|------|------|------|
| **压抑的悲伤** | 表面平静，但眉头微微皱着 | 看着远方，没有焦点 | 轻轻抿着 |
| **强忍眼泪** | 眼眶泛红，眼睛微微眯起来 | 努力看向别处，或者眨眼忍住 | 微微颤抖 |
| **释然** | 眉头舒展，有一丝淡淡的微笑 | 平静地看向前方 | 放松，微微上扬 |
| **孤独** | 表情空洞，没有明显情绪 | 看着某处但目光空的 | 微微张开，像在发呆 |
| **思念** | 表情柔软，带一点惆怅 | 看着"那个人曾经在"的地方 | 微微动，像要说什么又没说 |
| **期待** | 眼睛微微睁大 | 专注看向某个方向 | 微微张开 |
| **失落** | 眼神暗淡，眉毛耷拉 | 低头或看向地面 | 嘴角微微下垂 |
| **倔强** | 咬紧牙关，眉头紧皱 | 直视前方，不躲避 | 紧闭 |
| **如释重负** | 深呼吸，肩膀放松 | 闭眼或看向天空 | 长长呼出一口气 |

### 情绪 → 肢体

| 情绪 | 整体姿态 | 手 | 脚步 |
|------|----------|-----|------|
| **压抑** | 肩膀微塌，身体微微蜷缩 | 无意识握紧/松开 | 慢，拖着走 |
| **悲伤** | 低着头，身体收缩 | 抱着自己，或垂着 | 沉重 |
| **释然** | 肩膀放松，站直 | 自然垂下，或轻轻张开 | 轻盈 |
| **孤独** | 独自站着，和周围有距离 | 插口袋，或抱着东西 | 漫无目的 |
| **思念** | 微微侧身，像在等人 | 无意识触碰某物 | 原地不动 |
| **紧张** | 身体绷紧，微微前倾 | 攥紧，或不停动 | 小步，犹豫 |
| **决心** | 挺胸抬头，站稳 | 握拳，或用力抓住什么 | 坚定 |

### 动作参考

| 动作 | 传达的情绪 | 提示词写法 |
|------|------------|------------|
| **手贴玻璃窗** | 思念、触不到、隔阂 | 她把手轻轻贴在玻璃上，像想触碰窗外的什么 |
| **转身** | 告别、决定、离开 | 她慢慢转过身去，背对着镜头 |
| **抬头看天** | 释然、祈求、迷茫 | 她抬起头看向天空，眼神里有迷茫 |
| **低头** | 失落、逃避、沉思 | 她低下头，头发遮住了表情 |
| **闭眼** | 承受、回忆、放弃抵抗 | 她慢慢闭上眼睛，像在承受什么 |
| **深呼吸** | 平复情绪、准备、如释重负 | 她深深吸一口气，然后慢慢呼出来 |
| **擦眼泪** | 悲伤、但想坚强 | 她用手背快速擦过眼角 |
| **奔跑** | 逃离、追赶、释放 | 她突然跑起来，像在逃避什么 |
| **停下脚步** | 犹豫、意识到什么、放弃 | 她突然停住，站在原地不动了 |
| **回头** | 留恋、犹豫、不舍 | 她回过头，看向来时的方向 |
