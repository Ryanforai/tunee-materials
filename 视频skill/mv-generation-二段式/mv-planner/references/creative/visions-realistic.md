# visions · 写实系

> 真实场景，情绪/概念/内心世界的视觉外化。输入已给出每段画面意图，本文件定义如何把这些意图拍得更精致、更有层次。

## 概念表达原则

- 先从 `stage_plan` 提炼 1-2 个反复出现的核心意象
- 同一意象重复出现时，必须改变至少一项：载体 / 景别 / 光线 / 颜色 / 空间关系
- 不要把同一个技巧连用三次；概念片需要变奏，不需要口号重复

### 概念片的高级感来源

- 抽象概念必须落在具体载体上：皮肤 / 玻璃 / 水 / 金属 / 雾 / 布料 / 影子
- 一个意象先建立，再变奏，再升级，不要第一次出现就打到最满
- 真正厉害的 visions 不是"什么都怪"，而是"有一套稳定规则在慢慢失控"

---

## 写实系拍法

每个 stage 优先明确：

- 一个情绪锚点
- 一个主要视觉载体（人 / 物 / 光 / 反射 / 空气）
- 一个主镜头策略
- 一个呼吸细节

### 写实系怎么拍贵

- 先让空间真实成立，再往里塞概念
- 概念优先落在反射、风、布料、水痕、屏幕、玻璃上
- 让人物像情绪的承载体，而不是剧情说明员

### 表现原则

1. 情绪优先于逻辑，但仍然服从输入里的 stage 顺序
2. 人物是情绪容器，不必做复杂事件，但必须有明确的身体状态
3. 重复不是照抄，同一意象每次都要换语境、角度、光线或距离
4. 副歌段把视觉密度推到最强，但仍保留可拍的现实质感

### 镜头风格

**精致写实（默认）：**
- 运动：缓推 / 固定 / 轻柔跟随 / 滑轨 / 缓慢环绕
- 人物：允许慢动作感、重复动作、仪式感动作
- 场景：允许半抽象（真实空间 + 概念化光线/反射/材质）

**纪录写实：**
- 运动：手持晃动 / 跟拍 / 固定窥视
- 场景：让真实空间自己说话，不做过重装饰

---

## 光色层次规则

- 主色必须清楚，辅色只负责提醒或刺点
- 辅色优先落在：背景边缘 / 反射面 / 玻璃 / 水面 / 金属 / 雾气
- 不同 stage 如果是同一情绪家族，主色可稳定，辅色变化
- 不同 stage 如果是明确对立，主色就要切换，而不是只改一个形容词

### 廉价梦境警报

以下组合很容易让 visions 掉到"AI 梦境拼贴"：

- 每段都极端，没有基线
- 每段都只靠发光、漂浮、烟雾
- 没有稳定主色，只是一堆好看颜色
- 每段都在说"神秘"，但没有具体载体

---

## Stage 级变奏

- 同一人物在不同 stage 中，至少变化一项：景别 / 视线关系 / 空间层次 / 光色对比 / 材质感
- 同一空间在不同 stage 中，至少变化一项：光源位置 / 反射载体 / 空气质感 / 色偏 / 前景密度
- 副歌、drop、崩塌点要把概念推到最强，但仍须与当前分支的世界规则一致

### 强视觉峰值

最强段优先靠这些东西建立记忆点：

- 不合理但清楚的空间比例
- 一个人和巨大负空间的关系
- 材质突然变化但仍然可见
- 主色稳定，辅色像刺一样打进来
- 焦点从人移到触发物，或反过来

### 反重复

- 同一情绪不能连续三段都靠同一种技巧完成
- 如果前一段用了脸部特写，这一段优先换到空间、反射、背影或身体局部

### Case：UNFORGIVEN — 多人写实概念

> visions · 写实系 | 色调: 胶片复古 + 当代简约混搭 | K-pop
> **参考价值：** 多人物队形构图、日夜场景强对比切换、战损妆容特写、色调三段递进（户外自然光→琥珀暖光→冷蓝夜景+车灯）

**summarize**：以西部/机能废土美学为视觉骨架。白天户外停机坪五人同步齐舞，室内昏暗通道战损妆容极端特写，夜晚街道五人手牵手站立。色调沿「户外自然光→复古琥珀暖光→冷蓝夜景+车灯白光」三段递进。全片没有故事线，只有站姿、表情、队形变化。
