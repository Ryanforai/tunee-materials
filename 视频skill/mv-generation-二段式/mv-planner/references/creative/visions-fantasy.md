# visions · 纯幻想

> 完全架空世界观，不遵循现实物理逻辑。输入已经给出每段要拍什么，本文件负责把这些段落放进一个自洽又高级的幻想世界。

## 概念表达原则

- 先从 `stage_plan` 提炼 1-2 个反复出现的核心意象
- 同一意象重复出现时，必须改变至少一项：载体 / 景别 / 光线 / 颜色 / 空间关系
- 不要把同一个技巧连用三次；概念片需要变奏，不需要口号重复

### 概念片的高级感来源

- 抽象概念必须落在具体载体上：皮肤 / 玻璃 / 水 / 金属 / 雾 / 布料 / 影子
- 一个意象先建立，再变奏，再升级，不要第一次出现就打到最满
- 真正厉害的 visions 不是"什么都怪"，而是"有一套稳定规则在慢慢失控"

---

## 世界规则

最晚在第一个完整 reveal 的 stage 内明确：

1. 重力规则：正常 / 反转 / 无重力 / 局部异常
2. 空间规则：有限 / 无限延伸 / 多空间并存
3. 光的规则：自然光源 / 无来源光 / 发光物体
4. 材质规则：真实 / 水晶 / 气体 / 液态金属 / 人工材质

规则一旦建立，贯穿全片，不得中途无故换系统。

### Stage 级推进

- 早段：建立规则，让观众读懂世界
- 中段：把同一规则推向更复杂的空间或更极端的构图
- 高潮段：视觉密度最高，但仍然服从已建立的规则
- 收束段：让世界规则松弛或回响，而不是突然变回现实

### 纯幻想怎么避免廉价

- 不要每一帧都塞满奇观，要给尺度和静止
- 一个奇观画面只保留 1 个支配规则
- 真正高级的幻想感来自秩序，不来自乱

### 异质化策略（最多 1-2 种，全片一致）

**比例失调：**
- `human figure dwarfed by architecture ten times their height`
- `ordinary objects scaled to building size`
- `reverse scale: enormous person in miniature environment`

**物理违背：**
- `explosion frozen mid-blast, glass shards suspended mid-air`
- `multiple time speeds: foreground in slow motion, background normal`
- `gravity reversed for one object only while everything else is normal`

**非人感：**
- `skin becoming translucent, veins and bones visible beneath`
- `fingers dissolving into crystalline particles at the tips`
- `hair behaving like liquid, moving against wind direction`

### 幻想光源词库

- `Sourceless ambient light emanating from the ground itself`
- `Multiple moons casting overlapping soft shadows`
- `Light that bends around corners in non-Euclidean paths`
- `Objects themselves glowing with internal cold luminescence`

## 光色层次规则

- 主色必须清楚，辅色只负责提醒或刺点
- 辅色优先落在：背景边缘 / 反射面 / 玻璃 / 水面 / 金属 / 雾气
- 不同 stage 如果是同一情绪家族，主色可稳定，辅色变化
- 不同 stage 如果是明确对立，主色就要切换，而不是只改一个形容词

### 廉价梦境警报

以下组合很容易让 visions 掉到"AI 梦境拼贴"：

- 每段都极端，没有基线
- 每段都只靠发光、漂浮、烟雾
- 没有稳定主色，只是一堆好看颜色
- 每段都在说"神秘"，但没有具体载体

---

## Stage 级变奏

- 同一人物在不同 stage 中，至少变化一项：景别 / 视线关系 / 空间层次 / 光色对比 / 材质感
- 同一空间在不同 stage 中，至少变化一项：光源位置 / 反射载体 / 空气质感 / 色偏 / 前景密度
- 副歌、drop、崩塌点要把概念推到最强，但仍须与当前分支的世界规则一致

### 强视觉峰值

最强段优先靠这些东西建立记忆点：

- 不合理但清楚的空间比例
- 一个人和巨大负空间的关系
- 材质突然变化但仍然可见
- 主色稳定，辅色像刺一样打进来
- 焦点从人移到触发物，或反过来

---

## 专属禁止词

`realistic lighting` / `natural daylight` / `documentary feel` / `handheld` / `raw` / `grounded`
`glowing` / `luminous` / `radiant`（用 color-gel-lit / backlit / side-lit 替代）
