# visions · 轻超现实

> 真实场景为基底，幻想元素作为"入侵者"渗透。输入已定义每段意图，本文件负责决定异常如何更高级地出现。

## 概念表达原则

- 先从 `stage_plan` 提炼 1-2 个反复出现的核心意象
- 同一意象重复出现时，必须改变至少一项：载体 / 景别 / 光线 / 颜色 / 空间关系
- 不要把同一个技巧连用三次；概念片需要变奏，不需要口号重复

### 概念片的高级感来源

- 抽象概念必须落在具体载体上：皮肤 / 玻璃 / 水 / 金属 / 雾 / 布料 / 影子
- 一个意象先建立，再变奏，再升级，不要第一次出现就打到最满
- 真正厉害的 visions 不是"什么都怪"，而是"有一套稳定规则在慢慢失控"

---

## 轻超现实设计

每个 stage 明确四件事：

- 现实基底：这一段真实世界长什么样
- 异常载体：异常落在物件 / 反射 / 水面 / 灯光 / 影子 / 材质的哪里
- 边界状态：现实和异常是混在一起，还是明显对立
- 阶段升级：比上一段更近、更强、更不可忽视在哪里

### 轻超现实怎么拍得高级

- 最开始要足够像现实，异常才会有杀伤力
- 异常第一次出现最好是小而准，不要大而空
- 越靠近高潮，异常越应该靠近人物身体或视线

### 异常升级梯度

按顺序升级，至少命中 3 步：

1. 真实基底建立
2. 单一小异常（一个物理细节）
3. 异常换载体（物件 → 反射 / 水面 / 灯光）
4. 异常靠近人物（开始回应人物）
5. 阈值瞬间（异常突然停止 / 归位 / 爆出反差）

禁止：

- 连续 3 shot 同一异常载体
- 只有情绪升级，没有视觉升级
- 一开始就完全脱离现实

### 异常载体词库

**物件：** `a glass of water stays perfectly still during an earthquake` / `a clock with no hands still ticking` / `a candle burning upward and downward simultaneously`

**反射：** `reflection shows a different time of day` / `mirror shows a gesture not being made` / `water reflection shows a figure that isn't there`

**影子：** `shadow facing the wrong direction` / `shadow slightly ahead of the person` / `shadow refusing to follow movement`

**灯光：** `light source color shifts without physical explanation` / `a single spotlight with no origin`

**材质：** `fabric rippling in still air` / `stone wall breathing slowly` / `glass floor showing depth that isn't there`

### 幻想元素写作规范

必须说明：

- 物理形态
- 如何出现
- 与现实的边界

不要只写"诡异""奇怪""不对劲"。

### 幻想光源词库

- `Light with no visible source, color-shifted` — 异常进入
- `Shadow moving against the light direction` — 影子异常
- `Water reflection brighter than the water itself` — 反射异常

### 镜头风格

- 现实段：缓推 / 固定静观
- 异常段：漂浮感推进 / 轻微扭曲；运镜变化本身就是切换信号
- 光线：现实符合物理；幻想可无来源光，但必须和现实形成明确色温或对比差

---

## 光色层次规则

- 主色必须清楚，辅色只负责提醒或刺点
- 辅色优先落在：背景边缘 / 反射面 / 玻璃 / 水面 / 金属 / 雾气
- 不同 stage 如果是同一情绪家族，主色可稳定，辅色变化
- 不同 stage 如果是明确对立，主色就要切换，而不是只改一个形容词

### 廉价梦境警报

以下组合很容易让 visions 掉到"AI 梦境拼贴"：

- 每段都极端，没有基线
- 每段都只靠发光、漂浮、烟雾
- 没有稳定主色，只是一堆好看颜色
- 每段都在说"神秘"，但没有具体载体

---

## Stage 级变奏

- 同一人物在不同 stage 中，至少变化一项：景别 / 视线关系 / 空间层次 / 光色对比 / 材质感
- 同一空间在不同 stage 中，至少变化一项：光源位置 / 反射载体 / 空气质感 / 色偏 / 前景密度
- 副歌、drop、崩塌点要把概念推到最强，但仍须与当前分支的世界规则一致

### 强视觉峰值

最强段优先靠这些东西建立记忆点：

- 不合理但清楚的空间比例
- 一个人和巨大负空间的关系
- 材质突然变化但仍然可见
- 主色稳定，辅色像刺一样打进来
- 焦点从人移到触发物，或反过来

---

## 专属禁止词

`surreal` / `dreamlike` / `otherworldly` / `uncanny`（必须写具体异常的物理形态）

## Case 参考

> case 只做风格校准和 treatment 参考。case **绝不能覆盖** 输入里的 `stage_plan`。

### Case：XOXZ — 多维空间 + 极端色调反差

> visions · 轻超现实 | 色调: 赛博暗黑 ↔ Y2K千禧梦幻极端反差 | K-pop
> **参考价值：** 多维空间嵌套、极端色调反差转场（深紫冷黑 ↔ 柔白淡黄）、单色沉浸空间、相同笑容复制的诡异感、时空倒流视觉效果

**summarize**：整片在两套色彩系统间切换：暗黑层以高对比深紫、品红、冷青主导；梦幻层转入柔和的白、天空蓝、淡黄。以沉睡少女为开头和结尾，中间在深空、废土卧室、黄色花海之间跳跃。结尾玻璃碎片倒退飞行回到原位，窗户复原，少女继续沉睡。
