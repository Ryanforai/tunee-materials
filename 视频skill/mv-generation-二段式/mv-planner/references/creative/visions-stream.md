# visions · 意识流

> 多时空 / 内心世界外化 / 人格并存。输入已给出每段意图，本文件定义如何把这些段拍成真正有层次的意识流，而不是随意乱跳。

## 概念表达原则

- 先从 `stage_plan` 提炼 1-2 个反复出现的核心意象
- 同一意象重复出现时，必须改变至少一项：载体 / 景别 / 光线 / 颜色 / 空间关系
- 不要把同一个技巧连用三次；概念片需要变奏，不需要口号重复

### 概念片的高级感来源

- 抽象概念必须落在具体载体上：皮肤 / 玻璃 / 水 / 金属 / 雾 / 布料 / 影子
- 一个意象先建立，再变奏，再升级，不要第一次出现就打到最满
- 真正厉害的 visions 不是"什么都怪"，而是"有一套稳定规则在慢慢失控"

---

## 意识流设计

每个 stage 内部先明确：

- `anchor`：这一段的触发物 / 触发动作 / 触发视线
- `layer`：现实层 / 记忆层 / 幻想层 / 恐惧层
- `light_signature`：这一层的独立光线和色温
- `repeated_action`：跨层重复的动作或姿态
- `hero_distortion`：这一段最强的意识偏移点

### 意识流怎么拍得高级

- 切层之前，先给触发物特写或明确视线
- 切层之后，立刻给出新的光线签名
- 最强的意识流不是乱，而是观众能感到"它为什么在这里断裂"

### 表现原则

1. 时空切换必须有触发物，不能无理由乱跳
2. 不同层要靠光线和色调清晰区分
3. 同一动作在不同层出现时，要形成镜像或变奏
4. 副歌可让多层并存，间奏和喘息段应减少层数

### 触发物词库

**视觉触发物（进入新层前给特写）：**
- 手持物：`a phone screen with an unread message` / `an old photograph` / `a key that doesn't fit any door`
- 反射：`a window reflection showing a different time of day` / `a mirror showing a gesture not being performed`
- 身体：`fingers slowly wrapping around a glass` / `eyes closing against their own will`
- 环境：`a single lamp turning on in a dark room` / `a door left ajar with light bleeding through`

### 时空层光线规范

切换时必须明确层次：

```
[现实层] → 正常光线，色温中性（4500-5500K），对比度正常
[记忆层] → 过曝 / 偏暖（2700-3500K）/ 轻微虚焦 / gradual falloff
[幻想层] → 欠曝 / 偏冷（6500-9000K）/ 色彩偏移 / rapid falloff
[恐惧层] → 高对比 / 极暗 / 冷色（蓝青或绿冷）/ hard cutoff
```

**各层光源词库：**

- **现实层：** `Natural light from window, even and neutral` / `Single overhead fluorescent, flat and functional`
- **记忆层：** `Overexposed warm window light bleeding across skin` / `Golden hour light trapped in an interior, overly warm`
- **幻想层：** `Cold sourceless ambient, color-shifted blue-green` / `Underlit from below, cold and directional`
- **恐惧层：** `Single hard sidelight, rapid falloff, near total darkness` / `Hard overhead with deep dark shadows below eyebrows`

### 镜头风格

- 首选：主观视角 / 记忆感漂移 / 反射层叠
- 允许：多重曝光感运镜 / 时空间跳跃 / 焦点主动转移
- 景别：允许不遵循正常逻辑，尤其在触发细节处大胆靠近
- 禁止：整片光线统一（会失去层次）

### 气质词参考

`fragmented memory` / `overlapping consciousness` / `time collapse` / `interior monologue` / `dissociation` / `psychological space` / `suspended moment` / `echo of a past self` / `involuntary recall`

## 光色层次规则

- 主色必须清楚，辅色只负责提醒或刺点
- 辅色优先落在：背景边缘 / 反射面 / 玻璃 / 水面 / 金属 / 雾气
- 不同 stage 如果是同一情绪家族，主色可稳定，辅色变化
- 不同 stage 如果是明确对立，主色就要切换，而不是只改一个形容词

### 廉价梦境警报

以下组合很容易让 visions 掉到"AI 梦境拼贴"：

- 每段都极端，没有基线
- 每段都只靠发光、漂浮、烟雾
- 没有稳定主色，只是一堆好看颜色
- 每段都在说"神秘"，但没有具体载体

---

## Stage 级变奏

- 同一人物在不同 stage 中，至少变化一项：景别 / 视线关系 / 空间层次 / 光色对比 / 材质感
- 同一空间在不同 stage 中，至少变化一项：光源位置 / 反射载体 / 空气质感 / 色偏 / 前景密度
- 副歌、drop、崩塌点要把概念推到最强，但仍须与当前分支的世界规则一致

### 强视觉峰值

最强段优先靠这些东西建立记忆点：

- 不合理但清楚的空间比例
- 一个人和巨大负空间的关系
- 材质突然变化但仍然可见
- 主色稳定，辅色像刺一样打进来
- 焦点从人移到触发物，或反过来

---

## 专属禁止词

`linear narrative` / `clear cause and effect` / `single timeline` / `realistic continuity` / `one location only`

## Case 参考

> case 只做风格校准和 treatment 参考。case **绝不能覆盖** 输入里的 `stage_plan`。

### Case：MONOCHROME-BLOOM — 黑白意识流 + 高饱和爆发

> visions · 意识流 | 色调: 黑白高反差 → 高饱和糖彩渐变 | J-pop / 电子流行
> **参考价值：** 黑白画面中局部插入高饱和彩色（凝胶灯/亚克力道具）、眼睛-屏幕-镜面作为反复出现的视觉元素、从黑白硬切到全彩的转场手法、角色在独立纯色背景前依次出现、高潮后回到自然光收尾

**summarize**：以黑白高反差为基底，前半段只在凝胶灯、有色亚克力道具、彩色织物里出现少量高饱和色。副歌时白光吞没画面后硬切进入全彩纯色背景空间。反复出现的视觉元素：眼部特写、电脑通话画面、相机取景框、粉盒镜面。高潮段每个角色站在独立配色的纯色背景墙前。结尾回到妆面特写和日落侧脸。
