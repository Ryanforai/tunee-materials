# 导演 Grammar

> story_mode / visions 专用。lip-sync 不加载本文件。

## 工作顺序

在写任何 treatment 之前，先做 3 层决策：

1. **全片导演规划**：这支 MV 整体怎么拍
2. **每段 stage treatment**：每个 stage 怎么拍得更强
3. **Shot 组织**：stage → SHOT 的合并/拆分/时间分配

不要直接从歌词跳到 treatment。

---

## 一、全片导演规划

### 从感受出发

MV 的第一使命是**传递感受**，不是讲清楚一件事。

正确的起点：**这首歌让人感受到什么，怎么把这种感受建成一个视觉世界。**

- "很伤感" → 太模糊
- "深夜一个人坐在厨房，突然不知道自己在等什么" → 这就有了

全片导演规划从这个具体感受出发，再决定空间、光、色彩、意象。

### 视觉世界四支柱

在做任何拍法决策之前，先确定四件事：

1. **空间**：这首歌发生在哪里？不是"在城市"，而是"城市里一条快被遗忘的老街"。空间决定基础情绪——空旷=渺小，密闭=压抑，边界=迷茫
2. **光**：用什么光线作为情绪底色？柔散自然光=温柔可触碰，强侧光=内心矛盾，逆光/剪影=孤独不可言说
3. **色彩**：3-5 个主色。蓝灰冷调=疏离，琥珀暖调=记忆与逝去，高饱和=当下外放，消色低饱和=沉重内敛。全片统一，关键转折处做色彩突变——这是最有力的视觉转场
4. **意象（motif）**：选 1-2 个贯穿全片的视觉元素。意象像音乐旋律——第一次出现让人注意到，第二次让人开始思考，第三次让人突然理解了什么。好的意象不解释自己

### 意象成长框架

意象不只是"反复出现"——它必须**成长**，每次出现带来不同含义：

```
意象第一次出现：建立（观众注意到它）
意象第二次出现：变奏（同一意象，不同语境/景别/光线/状态）
意象第三次出现：完成（含义翻转或深化，观众突然理解）
```

**示例 — 一扇门：**
- 第一次：紧闭的，人站在外面 → 被排斥
- 第二次：半开的，光从缝里透出来 → 希望
- 第三次：打开的，人走进去 → 到达

**示例 — 一双手：**
- 第一次：抓握着某个物件不放
- 第二次：松开了，手指悬在半空
- 第三次：放下后垂在身侧，但指尖还在微微颤

在全片导演规划中，必须确定：
- **motif_item**：选什么意象（手/门/水/光/镜子/某个物件）
- **motif_arc**：三次出现分别在哪些 stage，每次的状态变化是什么

### 全片规划清单

先从完整 `stage_plan` 得出一条全片拍摄曲线，而不是一段一段孤立书写。

至少内部明确：

- `core_feeling`：这首歌最核心的感受是什么（一句具体的话，不是"伤感"）
- `visual_thesis`：整片最重要的视觉母题
- `motif_item` + `motif_arc`：1-2 个意象 + 每次出现的状态变化
- `energy_curve`：哪几段克制，哪几段释放
- `light_curve`：色温、对比度、主光方向如何推进
- `hero_stages`：哪些段必须出记忆点
- `breath_stages`：哪些段必须留呼吸
- `contrast_points`：哪些相邻段需要明显反差

### 高级感来源

全片不是靠"更多元素"变高级，而是靠更清楚的支配关系：

- 一个段落只有一个主动作或主姿态
- 一个画面只有一个主光逻辑
- 一个镜头只有一个主要情绪方向——由光源变化、身体姿态递进、景别压缩或环境改变具体体现
- 一个高潮只靠 1-2 个强记忆点，不靠堆满技巧

`shot intention`（拍什么）和 `shot progression`（怎么成立）是两层。若最后只剩一句"缓慢推近人物"或"镜头拉远露出空间"，说明只写了运镜，没有写出 progression。

### 廉价感警报

以下情况连续出现 2 次以上，基本会明显掉档：

- 连续中景正面表演，没有景别压缩和释放
- 每段都只是"人物看镜头唱/演"，没有阈值变化
- 光线很好听，但没有明确光源和主次
- 所有段都在"用力"，没有呼吸和留白
- 每段都在炫运镜，反而没有一个能截图的定格
- 每段都把信息说满，观众没有补完空间

## 二、声画关系 + 音乐结构映射

### 声画关系（全片规划时确定，逐段可调）

画面和音乐的关系决定了 MV 的表达深度。有三层：

| 类型 | 定义 | 效果 | 常见陷阱 |
|------|------|------|---------|
| **声画同步** | 画面直接配合音乐情绪和节奏。歌词说雨就出现雨，副歌爆发画面也爆发 | 最直接，大部分商业 MV 的默认选择 | 上限低，容易变成"歌词字幕配图" |
| **声画分立** | 画面和歌词各自独立，在主题上共鸣而非直译。歌词说爱情，画面是空无一人的街道 | 给观众留解读空间，情绪更深 | 做不好会让人觉得画面和歌不搭 |
| **声画对位** | 画面和音乐故意形成反差。欢快音乐配冷漠画面 | 最难但最有冲击力——反差本身在说话 | 过度使用会让人困惑 |

**推荐策略**（一支 MV 可混用三种）：
- **Verse** → 声画分立（建立情绪空间，不直译歌词）
- **Chorus** → 声画同步（情绪释放，画面跟着走）
- **Bridge** → 声画对位（最深的那句话，用反差说出来）

在全片导演规划中，必须确定 `sound_image_strategy`：每个结构段的声画关系类型。

### 音乐结构映射表

`音乐结构` 是镜头节奏控制器，不是标签。

| 结构 | 强度 | 光线策略 | 声画关系倾向 | 核心指令 |
|------|------|---------|------------|---------|
| Intro | 低 | 单光源留暗 | 分立 | 第一帧就要抓力，但不打满 |
| Verse | 低→中 | 克制侧光 | 分立 | 让人物和场景先成立；不要直译歌词 |
| Pre-chorus | 中→高 | 空间压缩 | 分立→同步 | 蓄力但不释放 |
| Chorus / Drop | 高 | 强对比 / 强色温 | 同步 | 至少 1 个截图即成立的画面 |
| Bridge | 转向 | 反差光线 | 对位 | 画面和音乐故意做反差——最深的那层意思 |
| Outro | 余震 | 松弛 | 分立 | 保留改变后的空气感；留白让观众完成 |

## 三、Treatment 写法

### Treatment 必须携带的信息

每条 treatment 必须包含以下维度，generator 将直接执行这些指令：

1. **视觉落点清单**：不同观察主体的「观众发现」，用 ①②③... 编号。落点必须来自不同类别（人物动作/人物局部/物件/环境/光影），每个落点是 generator 的一个 beat 素材
2. **身体状态标注**（有角色的 SHOT 必须同时给出两层）：
   - `(body_frame: ...)` → **静帧定格**，generator 直接用于 frame_prompt ⑩ 神态模块。写面部/肢体可拍的定格状态（如 `jaw set, lips barely parted, chin lowered`）
   - `(body_arc: ...)` → **运动弧线**，generator 直接用于 video_prompt beat 的角色动作描述。格式必须是「起始状态 → 触发或变化 → 结束状态」（如 `shoulders drawn inward → single slow breath → drop 1cm and hold`）
   - 两层均禁止情绪泛词（`sad / happy / nervous / contained` 等不可拍的心理描述）
3. **氛围**：光线方向 + 情绪底色
4. **→衔接**

> **stage 是片段，不是镜头（shot）。** planner 将 stage 组织成 SHOT（含合并/拆分），然后从输入画面描述中提炼视觉落点。每个落点对应 generator 的一个 beat 素材。如果只写一条连续动作，generator 只能把这个动作切成时间片段，导致每个 beat 都在看同一个人。
>
> **不指定焦段、景别、运镜**——那是 generator 的工作。planner 只说「观众看到什么」，不说「用什么镜头拍」。

### 落点格式：必须是「运动事件」，不是「状态截面」

每条落点描述的是**这段时间里观众看到了什么变化**，不是「这里有什么」。

格式：`[起始状态] → [变化/动作] → [结束状态]`

至少两端须是具体的物理描述，变化过程必须可被摄像机拍到。

```
❌ 手停在琴键上方悬而不落                （状态截面，无变化轨迹）
✅ 手从键盖边缘滑入 → 指尖触键后微缩回 → 悬在琴键上方2cm不落

❌ 纱帘街灯斜纹在地面缓慢移动            （模糊的持续状态）
✅ 纱帘被微风推动 → 地面斜纹偏移约20cm → 光斑扫过琴键表面后消失

❌ Kaiya坐琴旁身体微弓不动              （完全静止，无变化）
✅ Kaiya坐琴旁 → 随一次缓慢呼吸上半身微微起伏 → 落回原位，肩线比进入时低了一点
```

### 落点数量

按 shot 时长弹性调整：

| shot 时长 | 落点数 | generator beat 数 | 关系 |
|-----------|-------|------------------|------|
| 3-5s      | 2-3   | 2-3              | ≈1:1 |
| 6-8s      | 3-4   | 3-4              | ≈1:1 |
| 9-12s     | 4-5   | 4-6              | 留 1 个收束 beat 给 generator |

**落点选择规则：**
- 6s 以上的 shot，落点中至少包含 1 个非人物类别（物件/环境/光影），为 generator 的呼吸镜头提供素材
- 避免所有落点都围绕同一人的不同动作——那只是"一个表演的分镜"，不是"多个发现"

### Hero Frame 标注

在 hero frame 所在 SHOT 的 treatment 末尾追加 `[hero]`。

### Treatment 原则

- 输入画面描述中的多个视觉元素 → 提炼为不同类别的落点（人物/局部/物件/环境/光影）
- 不指定焦段、景别、运镜——那是 generator 的决策
- 不允许改掉 stage 本来的主动作、角色、场景和段落意义
- 不允许把多元素片段压缩成一条连续动作——那会让 generator 只能做时间切片

### Treatment 快速检查

每个 stage 写 treatment 前，先问 4 个问题：

1. 输入画面描述里有几个不同的观察对象？（分别属于什么类别）
2. 这一段最值得被记住的一瞬间是什么
3. 这一段的光是"照亮什么"，不是"照亮一切"
4. 这一段和上一段相比，真正新增的视觉信息是什么

### 写法示例

好的 treatment（SHOT 格式 + 运动事件落点 + body_frame/body_arc + 氛围 + 衔接）：

- SHOT_1 [0-8s] (S1): ①手从键盖边缘滑入 → 指尖触键后微缩回 → 悬在琴键上方2cm不落 ②Kaiya坐琴旁 → 随一次缓慢呼吸上半身微微起伏 → 落回，肩线比进入时低一点 (body_frame: eyes unfocused downward, lips sealed, jaw slack) (body_arc: chest rises in slow breath-hold → drops, shoulders settle 1cm lower) ③纱帘被微风推动 → 地面斜纹偏移约20cm → 光斑扫过琴键后消失 ④琴键反射的光斑随纱帘回摆 → 从左往右游走 → 停在键盘边缘 | 克制·侧光压暗 →起身脚步引出下段
- SHOT_4 [18-26s] (S4): ①他从窗边走过 → 纱帘掠过前景遮住身体半侧 → 走至窗框停住 ②手指从窗框上沿起 → 沿木纹缓慢下划 → 中途停住，指节轻微收紧 (body_frame: jaw set, lips pressed, shoulders braced) (body_arc: hand slides down wood grain → finger presses once into the grain → hand stops, knuckles whiten slightly) ③纱帘投影在地面 → 随他移动而拉伸变形 → 他停下时投影也静止 ④窗外路灯在他肩上 → 光带随他右移 → 他停下时光带压在右肩不动 | 压抑·窗外逆光 →硬切回室内形成呼吸断裂 [hero]
- SHOT_7 [38-43s] (S7): ①门缝透入的光线 → 随轻微气流宽窄变化约5mm → 回到初始宽度 ②地面灰尘在光柱内 → 从静止开始缓慢上浮 → 漂至光柱边缘后消散 ③远处走廊 → 脚步声由远及近 → 声音在门外停住，人未出现 (body_frame: brows knitted, chin lifted, lips sealed) (body_arc: N/A — 空镜，无角色) | 等待·单光源 →对比前段的拥挤 [motif:门·第二次] [声画分立]
- SHOT_10 [58-70s] (S10): ①指尖从木纹起点划过 → 经过三道年轮纹 → 在最末一道停住，像写完最后一笔 ②木纹年轮 → 从模糊到清晰对焦 → 停在清晰后保持静止 ③他的眼睛从低垂 → 缓慢抬起 → 停在某个画外方向，眼眶轻微泛红 (body_frame: heavy-lidded eyes, soft unfocused gaze, brows slightly raised) (body_arc: hand slides to stop → lifts off wood surface → hangs in air, fingertips trembling faintly) ④手从木面抬起 → 悬在半空 → 指尖在空中轻微颤动后静止 ⑤窗外天色 → 从深蓝渐变为灰白 → 在他抬头时完成最后色变 | 收束·散射柔光 →抬头动作接下段远景

## 四、蒙太奇策略 + 反差与反重复

### 蒙太奇思维

蒙太奇的本质：**A + B = C，而 C 是 A 和 B 各自都没有的意思。** 两个镜头放在一起，产生第三个含义。

反差/反重复是**防守**（避免平庸）。蒙太奇是**进攻**（主动构建意义）。两者都要做。

#### 四种蒙太奇策略

| 类型 | 做法 | 适用场景 | 在 treatment 中如何标注 |
|------|------|---------|----------------------|
| **节奏蒙太奇** | 剪辑节奏跟随音乐，但不是"每拍切一次"——张弛对比才制造节奏感（一个镜头撑5s，然后连续5个切在2s内） | Chorus / Drop 的快切段 | treatment 内标注节奏密度 |
| **对比蒙太奇** | 相邻 shot 并排对立画面：过去/现在，喧嚣/安静，大/小，快乐表情/孤独环境 | 需要情绪碰撞的 stage 交界处 | `→对比:` 标注对位关系 |
| **隐喻蒙太奇** | 插入自然界或物件画面暗示情感：花枯萎=爱消逝，水涨=情绪漫溢，鸟飞=自由或离去 | 呼吸镜头 / bridge 段 | treatment 中描述隐喻意象 |
| **积累蒙太奇** | 同一主题画面反复出现逐渐加强，直到临界点。第三次的重量和第一次完全不同 | motif 成长弧线的三次出现 | 配合 motif_arc 使用 |

**注意**：不需要每个 stage 都用蒙太奇。大部分 stage 正常拍，但全片至少有 2-3 处主动运用蒙太奇策略构建"第三层意义"。

### 反差

全片至少 2 次明显反差跳跃，优先从这些维度做：

- 景别：远 → 近 / 近 → 远
- 动静：完全静止 → 明显动作
- 色温：冷 → 暖 / 暖 → 冷
- 空间：封闭 → 开阔 / 室内 → 室外 / 现实 → 异常

### 反重复

相邻 shot 不要只换说法。

若相邻两段同时满足以下三项，基本等于无效重复：

1. 同一场景
2. 同一情绪段位
3. 同一构图和镜位关系

至少推进一项：

- 新空间阈值
- 新物件状态
- 新身体姿态
- 新关系距离
- 新光线状态
- 新情绪重量

### 反重复轮转表

```
上一 shot 主要特征 → 本 shot 优先方向:
  正脸人像   → 背影 / 侧面 / 局部 / 空镜 / 反射
  固定镜头   → 运镜（推 / 拉 / 环绕 / 跟随）
  暖色调     → 冷色或中性
  近景       → 全景或特写（跳过中景）
  人物独占   → 加入环境 / 前景 / 物件
  高强度     → 呼吸 / 留白
```

## 五、Hero Frame

每支 MV 至少 2-3 个强定格画面，优先落在：

- 开场第一记抓人画面
- 副歌开头
- 明确转折点
- 结尾余味段

优先类型：

- 逆光剪影 + 大面积留白
- 正中央孤立人物 + 对称空间
- 强单侧光 + 半明半暗的脸
- 低角仰拍或高角俯拍 + 极简动作
- 近景人像 + 前景遮挡 / 反射 / 玻璃层次

第一帧也要尽量具备 hero 质感，不要用平庸的默认姿态开场。

### Hero Frame 判断法

一个 hero frame 至少满足其中 3 条：

- 轮廓一眼能认
- 身体线条清楚
- 光源关系清楚
- 背景足够干净或足够有秩序
- 缩成缩略图也能成立
- 就算静音截图也有情绪

### 光线词库

**人造光源：** `A single lamp` / `phone screen` / `colored storefront sign` / `bare overhead bulb` / `A single tungsten floor lamp` / `Fluorescent overhead tubes`

**自然光源：** `Late afternoon sun through blinds` / `pre-dawn grey` / `overcast diffused` / `Golden hour sunlight through curtain gap`

**舞台光源：** `A single theatrical spotlight directly overhead`

**叙事功能光：**
- `Car headlight sweeping`：关系被打断、时间流动
- `Doorway flooding with light`：进入、离开、阈值变化
- `Phone screen spill`：私密、夜晚、一个人的情绪回响

**光线叙事功能：**
- 高对比强侧光 → 冲突 / 决断
- 柔和漫射 → 脆弱 / 停顿
- 背光逆光 → 离开 / 转折 / 成为剪影

**衰减速度：**
- `rapid falloff`：高对比、切断式情绪
- `gradual falloff`：柔和、延迟释放
- `hard cutoff`：戏剧最强、空间像被切开

**高级感：** 一段里只保留一个主光决定权。辅光只负责托住信息，不要和主光抢戏。

**辅色克制词：** `subtle` / `faint` / `barely perceptible` / `ghostly` / `whisper of`

### 禁止词

**精致写实：** `handheld shake` / `gritty` / `raw` / `unpolished` / `neon`

**纪录写实：** `smooth gimbal` / `perfectly centered` / `cinematic lighting` / `rim light` / `god ray`

**通用：** `glowing` / `luminous` / `radiant` / `magical glow` / `floating particles` / `lens flare explosion` / `cool and trendy` / `edgy vibes`
