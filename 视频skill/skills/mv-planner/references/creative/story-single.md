# story_mode · 单线

> 输入已经给出单线剧情的 stage 规划。本文件只负责把这条线拍得更有戏、更有层次、更像导演作品。

## 风格判断

从完整 `stage_plan` 的人物关系、空间气质、音乐结构推断拍摄风格。`visual_style` 存在时优先。

### 精致写实（默认）

- 运动：缓慢推进 / 固定静观 / 轻柔跟随 / 精确滑轨 / 缓慢环绕
- 禁止：无理由手持晃动 / 急速运镜 / 为炫技而炫技
- 景别：丰富变化，但每次变化都服务叙事推进
- 景深：浅景深优先，长焦（85–135mm）压缩空间
- 光线：精心单侧布光，完整三层光（主光 + 衰减 + 补光）
- 人物：克制精准，每个动作都带叙事含义

### 纪录写实

- 运动：手持晃动 / 跟拍 / 固定窥视 / 偷拍感长焦
- 禁止：过于顺滑的稳定器语言 / 甜美广告式构图
- 景别：中景/近景为主，允许不完美构图
- 光线：自然光 / 环境光，允许局部不均匀

## 单线拍法

每个 stage 至少明确：

- 这段的不可逆动作是什么
- 这段的情绪增量是什么
- 这段最强的 hero frame 是什么
- 这段与上一段相比新增了什么

### 单线高级化方法

- 不要把"发生了什么"平着拍完，优先找出这段最贵的一帧
- 一个 stage 里只挑一个最值钱的动作，不要什么都拍
- 如果是情绪段，不要只拍脸；给空间、手部、物件、门槛

### 单线递进规则

- 开场段：先给视觉悬念，不要立刻把全部信息说完
- 积累段：用人物动作、空间阈值、物件状态推动，不靠抽象内心独白
- 转折段：必须通过可拍到的动作完成（`[PASS]` 打开纸袋 / 拿起钥匙 / 推门进入 / 把外套搭上肩；`[FAIL]` 只在 description 里说"她终于想通了"）
- 余波段：让空间和身体状态保留变化后的后劲

## 高级故事镜头的常见来源

- 阈值：门框、楼梯、车门、窗前、走廊转角
- 前景：玻璃、水汽、帘子、肩膀、门框边缘
- 关系距离：同框但不接触 / 接触但不对视 / 对视但不靠近
- 物件状态：钥匙、手机、衣领、杯子、镜子、信封
- 材质：湿地面、旧木头、金属反光、布料褶皱

这些东西不是装饰，是把普通剧情拍贵的方式。

## Stage 级推进规则

- 同一人物连续两个 stage，至少推进一项：景别 / 角度 / 身体姿态 / 前景层次 / 光线状态 / 空间深度
- 同一场景重复出现时，必须出现可拍到的变化：门是否打开、物件状态是否变化、人物距离是否改变、光线是否转向
- 副歌和转折段要明显更有抓力，但不能脱离输入里的主动作

### 同一场景内的推进

同一场景连续出现不等于重复。必须至少推进一项：

- 人物距离镜头更近或更远
- 人物从空间边缘进入中心，或反过来
- 物件状态发生变化
- 光线方向或强度变化
- 前景遮挡或景深层次变化

让同一场景拍出高级感：开始拍"人被空间吞住" → 中段拍"人和空间发生接触" → 结束拍"人改变了空间里的某个状态"

### 不要拍成电视剧 coverage

以下组合很容易让画面立刻掉到平庸：

- 连续三个 shot 都是正常视平中景
- 连续三个 shot 都只给正脸
- 每段都把动作解释一遍，没有留白
- 所有人物都站着把话或歌词"演完"

### Case：BLUE — Pop Ballad 恋爱短片

> story_mode · 单线 | 色调: 胶片复古暖调 | Acoustic pop / Ballad
> **参考价值：** 环境先于人物出场、自然元素（风/水/鸟群）制造画面运动、双人关系靠简单动作成立、近景人像与远景海天交替、痕迹式收尾（脚印/背影）

**summarize**：海边落日恋爱短片。前段用车窗、湖面、长椅、树影建立空间，女主延迟出场。中段用牵手奔跑、海浪嬉闹、沙滩自拍三组动作完成情侣关系。全片画面运动由风吹头发、海浪打脚、鸟群飞过承担，不需要复杂表演。结尾不用正脸，只留脚印、背影。
