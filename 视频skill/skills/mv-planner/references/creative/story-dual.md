# story_mode · 双线

> 输入已经给出双线 stage 规划。本文件定义如何把两条线拍出对位、反差和汇聚感。

## 风格判断

从完整 `stage_plan` 的人物关系、空间气质、音乐结构推断拍摄风格。`visual_style` 存在时优先。

### 精致写实（默认）

- 运动：缓慢推进 / 固定静观 / 轻柔跟随 / 精确滑轨 / 缓慢环绕
- 禁止：无理由手持晃动 / 急速运镜 / 为炫技而炫技
- 景别：丰富变化，但每次变化都服务叙事推进
- 景深：浅景深优先，长焦（85–135mm）压缩空间
- 光线：精心单侧布光，完整三层光（主光 + 衰减 + 补光）
- 人物：克制精准，每个动作都带叙事含义

### 纪录写实

- 运动：手持晃动 / 跟拍 / 固定窥视 / 偷拍感长焦
- 禁止：过于顺滑的稳定器语言 / 甜美广告式构图
- 景别：中景/近景为主，允许不完美构图
- 光线：自然光 / 环境光，允许局部不均匀

## 双线识别

- 每个 shot 必须明确属于线 A 或线 B
- **treatment 开头必须标注 `[线A]` 或 `[线B]`**
- 若 stage_plan 已交替推进，保持这种交替
- 不要为了形式强行造假交替；以输入顺序为准

## 双线关系类型

- **对位型**：一条线爆发，另一条线压抑
- **平行型**：两条线同步推进，但空间不同
- **汇聚型**：越接近结尾，两条线视觉差异越小

## 视觉区分

两条线须有可识别差异，至少稳定保留两项：

- 色温：线 A 偏暖 / 线 B 偏冷（或反之）
- 景别倾向：一条多中近景，另一条多远景
- 运镜风格：一条偏跟随，一条偏固定
- 场景类型：一条室内，一条室外；或一条开阔，一条封闭

### 高级双线不是"轮流播剧情"

切线时最好同时完成两件事：

- 给信息差
- 给情绪撞击

如果只是 A 讲一句，B 再讲一句，画面会很像普通叙事 coverage。

## 对位规则

两条线切换时，至少形成一项对位：

- 热 / 冷
- 动 / 静
- 靠近 / 回避
- 明亮 / 阴暗
- 开阔 / 压迫

## 汇聚规则

如果是汇聚型双线：

- 前段差异要清楚
- 中段开始出现呼应物件或呼应动作
- 结尾前后，色温、景别或构图可以逐步靠拢

## 双线对位方法

- **严格交替节奏：** stage 按 A/B/A/B 或 A/A/B/B 排列
- **物件呼应：** 两条线通过同一件物品建立联系（手机、外套、钥匙、未发出的消息）
- **色温对立：** 线A室外冷色 vs 线B室内暖色
- **动静对立：** 线A动态行走 vs 线B静止留守
- **汇聚型结尾：** 越接近结尾，两线色温靠拢、景别靠近

## 高级故事镜头的常见来源

- 阈值：门框、楼梯、车门、窗前、走廊转角
- 前景：玻璃、水汽、帘子、肩膀、门框边缘
- 关系距离：同框但不接触 / 接触但不对视 / 对视但不靠近
- 物件状态：钥匙、手机、衣领、杯子、镜子、信封
- 材质：湿地面、旧木头、金属反光、布料褶皱

## Stage 级推进规则

- 同一人物连续两个 stage，至少推进一项：景别 / 角度 / 身体姿态 / 前景层次 / 光线状态 / 空间深度
- 同一场景重复出现时，必须出现可拍到的变化：门是否打开、物件状态是否变化、人物距离是否改变、光线是否转向
- 副歌和转折段要明显更有抓力，但不能脱离输入里的主动作

**同一条线**前后两次出现时：至少推进一项（物件状态 / 新空间位置 / 新情绪节点 / 光线变化）

**两条线之间**切换时：至少形成一项清楚差异，不要像同一条线换了衣服

### 不要拍成电视剧 coverage

以下组合很容易让画面立刻掉到平庸：

- 连续三个 shot 都是正常视平中景
- 连续三个 shot 都只给正脸
- 每段都把动作解释一遍，没有留白
- 所有人物都站着把话或歌词"演完"

## Case 参考

> case 只做风格校准和 treatment 参考。case **绝不能覆盖** 输入里的 `stage_plan`。

### Case：PARALLEL — 都市双线离别

> story_mode · 双线（汇聚型） | 色调: 当代简约，线A偏冷 / 线B偏暖 | Pop / R&B
> **参考价值：** 严格 A/B 交替节奏、室外冷色 vs 室内暖色视觉区分、物件状态推进叙事（手机/外套/钥匙）、汇聚型结尾两线视觉趋同、不可逆动作完成转折

**summarize**：同一座城市同一个傍晚，两个人各自经历关系结束后的第一个小时。线A跟着离开的人穿过冷蓝街道和天桥——动态、行走、室外；线B跟着留下的人在暖琥珀色公寓里——静止、被遗留物件包围。两条线通过物件呼应（同一件外套、同一串钥匙、同一条未发出的消息）。越接近结尾，线A背景出现暖色窗光，线B开始向窗外看——色温逐渐靠拢。最后线B的人走到窗边俯看街道，线A的人刚从那条街走过已不在画面里。
