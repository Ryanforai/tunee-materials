# Prompt 写法指南 · Seedance 2.0（story_mode / visions）

> generator 的核心工作：把 planner 的导演 treatment 翻译成 Seedance 2.0 能理解的 video_prompt。

---

## 一、video_prompt 整体结构

```
[资源绑定]
@图片1是角色X的参考形象，@图片2是场景「Y」的参考图，保持画面风格与参考图一致；

[一致性锁定]
全片保持统一视觉风格，角色保持一致五官和服装，场景保持一致环境和光线，核心道具保持一致外观特征；

风格：[渲染风格/色彩基调/视觉调性，大气效果写在此行]
场景：[此刻这个空间的真实样子——环境+光线+正在发生的动态元素（风/花/动物/自然现象/室内的烟/光变化等），写物理状态，不写形容词]

[Xs-Xs] 镜头1（景别 · 镜头运动）
[视觉元素A + 动作] →
[视觉元素B + 动作] →
落点：[定格画面——描述可拍的物理状态]
镜头：[运动方式]，[触发条件]时锁定。

[Xs-Xs] 镜头2（景别 · 镜头运动）
[视觉元素A + 动作] →
[视觉元素B + 动作] →
落点：[定格画面——描述可拍的物理状态]
镜头：[运动方式]，[触发条件]时锁定。

...（按实际子镜头数继续）

约束：avoid jitter，avoid identity drift，avoid bent limbs；[特殊条目]
```

**子镜头划分规则（核心）：**

一个 SHOT 的时间范围内，**必须按时间段拆分为多个子镜头**，每个子镜头独立标注时间、景别、镜头运动。

- 每个子镜头时长目标：**2-4s**，不超过 6s
- **子镜头数量由 planner 的落点数量决定**，不用公式计算：planner 给几个落点，generator 展开几个子镜头。落点数量参考：4-5s → 2-3镜，6-8s → 3-4镜，9-12s → 4-5镜，13-15s → 5-6镜
- 相邻子镜头必须切换景别或镜头运动，禁止连续两个子镜头用相同景别+相同机位
- 子镜头内的 beat 密度由 DIRECTING 的 pacing 决定，不在时间轴行标注

**时间轴行格式：** `[start_time-end_time s] 镜头N（景别 · 镜头运动）`
- **时间从 0 开始计数**，每个 SHOT 独立重置，与全片绝对时间无关
- 例：SHOT 时长 9s → `[0-4s] 镜头1` / `[4-7s] 镜头2` / `[7-9s] 镜头3`，不写全片绝对时间
- 子镜头之间连续无间隙，最后一镜的 end_time = SHOT duration
- 景别：远景 / 全景 / 中景 / 近景 / 特写 / 极特写 / 微距 / 俯拍 / POV
- 音乐段落信息用于内部决策 beat 密度，**不写入时间轴行**

**语言：** `language_code = zh-CN` 用中文，其他用英文。禁止中英混杂（运镜术语和 body: 关键词除外）。

## 二、@图片N 编号规则

每个 shot 独立从 1 开始重新编号，**先角色后场景**：

- `@图片1` = 第 1 个角色，`@图片2` = 第 2 个角色（或单角色时的场景）
- 场景排在所有角色之后
- **空镜（character_name = null）：** 不写角色绑定，`@图片1是场景「Y」的参考图`，禁止写已注册角色名

**角色引用规则：**
- 注册角色：绑定后必须用 `@图片N` 指代，禁止"第二个人物"等泛称
- 临时配角：必须详细描述外貌（性别、年龄段、体型、服装、发型）
- 路人/群众：作为环境元素，不需详描

## 三、Beat 序列写法（核心）

Beat 序列是 video_prompt 的主体。每个 `→` 是**观众视线的一次落点**，来自 planner treatment 的 ①②③... 视觉落点。

**核心原则：不写"角色是什么状态"，写"观众的眼睛在这一刻落在哪里"。**

### Beat 写法六律

**① 每个 beat 只写一件事**
"转身微笑"是两件事，必须拆成两个 `→`。

**② 身体动作写传导顺序，不写结果**
```
❌ @图片1转身
✅ 肩膀先启动，躯干跟上，头是最后转完的，发丝惯性比头慢半拍才落定
```

人体传导规律（按需展开）：
- 转身：肩 → 躯干 → 头 → 发丝
- 伸手：肩 → 肘 → 腕 → 指尖
- 起身：腿 → 腰 → 脊背 → 肩 → 头
- 跪落：膝先弯 → 重心沉 → 裙摆最后展开

**③ 眼神写轨迹，不写状态**
```
❌ eyes forward，gaze steady
✅ gaze 从地面开始抬，抬到一半停住两拍，才完成抬起 snap 锁定镜头
```

**④ 每个 SHOT 至少一次速度突变**
全程匀速 = 人物呆滞。速度突变分布在整个 SHOT 范围内，不要求每个子镜头都触发。触发词：`骤然 / snap / 猛然 / 瞬间收住`
```
[慢慢积累] → 骤然收住 / snap 锁定 / 猛然转向
```

**⑤ 附属物写惯性延迟**
发丝、裙摆、袖口比身体慢——这一拍的时序差是真人感最直接的来源：
```
发丝比头慢半拍落定 / 裙摆动作收住后还荡了一下才垂下
```

**⑥ 落点是可截图的定格画面**
```
落点：手指完全握住，指节压白，花瓣落在手背不再滑落
```

### 三类 Beat 素材

**A. 身体局部**（制造真人感）
指尖骤然停住 / 锁骨随呼气轻微起伏 / 袖口随手臂翻起一角 / 重心偷偷从一脚转向另一脚

**B. 眼神序列**（制造存在感）
gaze 从地面滑动抬起 → 抬到一半停住两拍 → snap 锁定目标，不再动

**C. 情绪弧线**（一镜内情绪必须变化，不能一个状态撑完整镜）
初始状态 → 触发事件 → 情绪转变 → 落点表情

情绪身体化参考（禁止直接写情绪词，按需转译）：

| 情绪 | 眼部 | 嘴唇/下颌 | 身体信号 |
|------|------|---------|---------|
| 克制的悲 | gaze locked on fixed point, not blinking | lower lip pressed inward | one slow swallow |
| 即将哭泣 | eyes glistening, lower lashes wet | lips sealed but trembling | chin pulls down slightly |
| 交托/放松 | eyes close slowly | jaw releases | 深呼气肩膀落下半厘米 |
| 坚定 | eyes sharp, direct contact | lips press then release | chin lifts one degree |
| 隐秘认可 | gaze softens | lips press, not a smile | 指尖轻微收拢 |

## 四、快切组（Chorus / 高能段专用）

仅在音乐高能段使用，在 beat 序列中用【】标注：

```
[慢 beat 缓冲] →
【快切组，共Xs】
→ [部位A + 瞬间动作]
→ [部位B + 瞬间动作]
→ [部位C + 瞬间动作]
[慢 beat 缓冲 / 落点]
```

**四条规则：**
- 组内每个 beat 只写部位+动作，不写速度修饰
- **组内镜头运动必须锁定**，快切结束后才继续推
- 前后各留一个慢 beat 缓冲，不能直接进入或跳出快切组
- 组内部位必须交替，不能连续写同一身体部位

---

## 五、按音乐段落的密度策略

Beat 密度跟着音乐能量走，不是每镜标配快切。

| 音乐段落 | Beat 节奏 | 快切组 | 落点时长 |
|---------|---------|-------|---------|
| Intro | 极慢，1个/2s | 无 | 长，2s+ |
| Verse | 匀速，1个/1s | 无 | 中，1.5s |
| Pre-chorus | 渐快，1个/0.8s | 可选，放末尾蓄力 | 短，悬而未发 |
| **Chorus** | **1个/0.3-0.4s** | **1-2组，核心** | **强，1.5-2s** |
| Bridge | 视情绪定 | 通常无 | 视情绪定 |
| Outro | 极慢，1个/2s+ | 无 | 极长，3s+ |

**Beat 密度参考：**

| shot 时长 | beat 总数 | 快切组时长 |
|---------|---------|---------|
| 4-5s | 4-6个 | 无或0.8s |
| 6-8s | 6-9个 | 1-2组共1.5-2s |
| 9-12s | 9-13个 | 2组共2-2.5s |
| 13-15s | 13-17个 | 2-3组共3s |

## 六、镜头层

**格式：** `镜头：[运动方式]，[触发条件]时锁定。`

镜头运动与 beat 序列**必须分行写**，禁止混在同一句话里。
**快切组期间镜头运动锁定。** 快切的冲击来自 beat 密度，不是镜头运动。

---

### 运镜决策：先问三个问题

每个子镜头在确定运镜之前，先问：

**问题一：这一镜的情绪是收紧还是释放？**
- 收紧（克制、悬念、等待）→ 镜头抵抗情绪，反向运动或锁定；让观众感到张力
- 释放（爆发、到达、崩溃）→ 镜头顺情绪，加速或大幅位移；让观众跟着呼出气

**问题二：这一镜和下一镜的关系是什么？**
- 同一场景延续 → 镜头运动方向为下一镜提供动势，剪辑点有方向感
- 场景切换 → 镜头运动在切点前反向收束，为切换制造空隙
- 情绪对比 → 镜头语言故意与上一镜反向，用运镜本身制造蒙太奇意义

**问题三：这一镜里有没有「空间矛盾」可以利用？**
- 主体距离感与心理距离感是否可以分裂？（主体越近，心理越远 → Dolly Zoom）
- 焦点可以在不移动镜头的情况下完成叙事转移？（Pull Focus，前景→背景或反向）
- 有没有可以穿透的介质（玻璃/水面/镜面/烟雾）能制造不可能机位？

---

### 决策之后：写清楚镜头在做什么

确定运镜意图后，用术语或物理描述都可以——关键是**让人看得懂这个镜头在做什么、为什么这么动**。禁止把术语堆在一起充当描述。

```
❌ Slow Dolly In + Smooth + Cinematic + Intimate
✅ Slow Dolly In，从腰部高度推到面部，眼睛充满画面时锁定。
```

## 七、全局写作规则

### 大气连贯
从 DIRECTING「大气」行取值，写入每个 shot 风格行，可微调浓度，不换种类。

### 道具连贯
道具首次出现必须写完整固定描述（颜色+型号/形态+材质），后续 shot 保持一致。

### 空间连续性
同一场景连续 shot，首句锚定主体位置：
```
❌ "@图片1单手搭在方向盘上"
✅ "驾驶座视角，@图片1左手搭在方向盘十二点位"
```

### 收束（仅全片最后 shot 末尾）
```
[后期处理词 2-3个叠加]收尾，[张力宣言]。
后期处理词：暗角 / 胶片颗粒 / 色差 / 运动模糊 / 轻微过曝
```

---

## 八、禁止词

```
dramatic / epic / breathtaking / stunning / gorgeous / beautiful / magical
cinematic lighting（必须写具体光源）
neon / 霓虹（用 colored gel light / colored storefront sign 替代）
ambient warm light / warm light + cool light（无主次）
sad expression / happy face（写可拍的物理状态）
```

**物理矛盾禁止：** wide angle + shallow DoF / long shot + extreme bokeh

**特效禁止：** 禁止描述角色身体半透明或溶解。"融入背景"用强光从外部包裹轮廓替代。

---

## 九、约束行

每个 shot 末尾固定写入：
```
约束：avoid jitter，avoid identity drift，avoid bent limbs；
角色运动与镜头运动全程分离；@图片N五官服装全镜一致。
```

按需追加：
- 呼吸镜头：`无角色面部入画`
- 镜像/双人：`禁止生成完整第二人脸，仅渲染[部位]局部反射`
- 特效场景：`禁止渲染角色身体透明或变形，用强光覆盖轮廓替代`

## 十、质量校准案例

> 仅供校准 beat 序列写法品质，禁止复制剧情。
> 注意：时间轴从 0 开始计数，与全片绝对位置无关。

**Verse · 6s · 3个落点 → 3镜（胶片写实）：**

```
@图片1是角色Ren的参考形象，@图片2是场景「旧公寓走廊」的参考图，保持画面风格与参考图一致；
全片保持统一视觉风格，角色保持一致五官和服装，场景保持一致环境和光线，核心道具保持一致外观特征；

风格：ARRI Alexa Mini LF 50mm，Kodak 5219 胶片模拟，subtle film grain，走廊壁灯暖黄单光源，薄雾弥散。
场景：旧公寓走廊，地板蜡油印浅黄，壁灯灯罩泛黄灰尘，远端门半开透出一线天光。

[0-2s] 镜头1（微距）
走廊地板木纹上一枚硬币侧面反光，壁灯投下的暖黄光带在硬币边缘骤然切断 →
落点：硬币纹丝不动，光切边锐利，背景走廊完全虚化。
镜头：锁定硬币表面，镜头本身不动——悬念靠静止制造，不靠运动。

[2-4s] 镜头2（近景）
@图片1站在走廊中段，肩线微微向右倾，重心从左脚悄悄转向右脚 →
腰先动，肩跟上，头最后转向走廊远端门的方向 →
落点：侧脸轮廓收住，下颌线清晰，远端门缝天光在脸侧描出一道细光线。
镜头：Tilt Up 跟随身体传导节奏从脚部到脸部，头转完时锁定——镜头和身体共同完成一次动作。

[4-6s] 镜头3（全景）
走廊远端门缝天光骤然变宽——风从外面推进来 →
地面蜡油印表面细尘被气流扫起，顺光飘向@图片1方向 →
落点：@图片1剪影在走廊中，细尘在逆光中悬住，不升不落，门缝光线压稳。
镜头：Slow Pull Out，@图片1剪影在退后中越来越小——情绪收紧，镜头反向拉开距离强化孤立感，为下一镜切换制造空隙。

约束：avoid jitter，avoid identity drift，avoid bent limbs；角色运动与镜头运动全程分离；@图片1五官服装全镜一致。
```

**Chorus · 12s · 4个落点 → 4镜（含快切组）：**

```
@图片1是角色Suki的参考形象，@图片2是角色伴侣的参考形象，
@图片3是场景「屋顶」的参考图，保持画面风格与参考图一致；
全片保持统一视觉风格，角色保持一致五官和服装，场景保持一致环境和光线，核心道具保持一致外观特致；

风格：Sony A7S III 35mm f/1.8，Kodak Portra 400 胶片模拟，gentle bokeh，黄金时刻强侧逆光，粉尘微浮。
场景：楼顶，金红落日在两人身后，城市线条被逆光压成剪影，屋顶水泥地面暖色反光。

[0-3s] 镜头1（特写）
对焦两人并排站立的肩线，城市剪影完全虚化入框 →
@图片1肩线微微下沉一次，像是长时间忍住的呼气终于松开 →
落点：两人肩线并排收住，逆光将肩线边缘描出金边，背景城市剪影压在画面远端。
镜头：Orbit 从@图片1肩线外侧缓缓环绕至正侧面——不是为了展示空间，是为了让观众感觉在悄悄靠近两人之间的距离。

[3-6s] 镜头2（微距）
@图片2手指从身侧缓缓向@图片1方向移动，腕先动，指尖落后半拍 →
指尖轻触@图片1手背指节侧面，触碰点皮肤轻微压白 →
落点：两人手指接触点压住，皮肤压白清晰，背景完全虚化。
镜头：Pull Focus 从@图片2手腕切换到触碰点，镜头本身不动——用焦点转移完成叙事，比推镜更克制，触碰的重量全在焦点落下那一帧。

[6-9s] 镜头3（近景）
@图片1肩膀先收紧，腰跟上，骤然转向@图片2 →
【快切组，共1.5s】
→ @图片1手指反握，五指一根一根扣住@图片2手背
→ @图片2手掌微微翻转配合，掌纹清晰
→ 两人握住的手背上一粒粉尘落下贴住皮肤
落点：双手扣紧，指节压白，暖金逆光在手背描边。
镜头：快切组期间镜头运动锁定；落点帧完成后 Tilt Up 抬起，为下一镜切点提供向上的动势。

[9-12s] 镜头4（全景）
两人并排站在屋顶，城市逆光剪影在他们身后展开 →
落日色温骤然加深一度，整片天空从金红变为暗橙 →
落点：两人剪影在画面中央，握住的手轮廓可见，暗角渐起，胶片颗粒叠加。
镜头：Dolly Back + Crane Up，两人剪影在退后中反而压住画面中央——用空间矛盾制造「渺小却在一起」的感觉。

约束：avoid jitter，avoid identity drift，avoid bent limbs；角色运动与镜头运动全程分离；
@图片1与@图片2五官服装全镜一致；禁止生成完整第二人脸正面，仅侧脸/轮廓/剪影入画。
```

## 十一、自检清单

```
□ 时间轴从 0 开始计数，每个 SHOT 独立重置，最后一镜 end_time = SHOT duration，子镜头之间无间隙
□ 子镜头数量与 planner 落点数量匹配（4-5s:2-3镜 / 6-8s:3-4镜 / 9-12s:4-5镜 / 13-15s:5-6镜），不用公式计算
```
