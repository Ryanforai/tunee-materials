# 导演 Grammar · Seedance 2.0

> story_mode / visions 专用。

## 工作顺序

在写任何 treatment 之前，先做 3 层决策：

1. **全片导演规划**：这支 MV 整体怎么拍
2. **每段 stage treatment**：每个 stage 怎么拍得更强
3. **Shot 组织**：stage → SHOT 的合并/拆分/时间分配

不要直接从歌词跳到 treatment。

---

## 一、全片导演规划

### 从感受出发

MV 的第一使命是**传递感受**，不是讲清楚一件事。

正确的起点：**这首歌让人感受到什么，怎么把这种感受建成一个视觉世界。**

- "很伤感" → 太模糊
- "深夜一个人坐在厨房，突然不知道自己在等什么" → 这就有了

全片导演规划从这个具体感受出发，再决定空间、光、色彩、意象。

### 视觉世界四支柱

在做任何拍法决策之前，先确定四件事：

1. **空间**：这首歌发生在哪里？不是"在城市"，而是"城市里一条快被遗忘的老街"。空间决定基础情绪——空旷=渺小，密闭=压抑，边界=迷茫
2. **光**：用什么光线作为情绪底色？柔散自然光=温柔可触碰，强侧光=内心矛盾，逆光/剪影=孤独不可言说
3. **色彩**：3-5 个主色。蓝灰冷调=疏离，琥珀暖调=记忆与逝去，高饱和=当下外放，消色低饱和=沉重内敛。全片统一，关键转折处做色彩突变——这是最有力的视觉转场
4. **意象（motif）**：选 1-2 个贯穿全片的视觉元素。意象像音乐旋律——第一次出现让人注意到，第二次让人开始思考，第三次让人突然理解了什么。好的意象不解释自己

### 意象成长框架

意象不只是"反复出现"——它必须**成长**，每次出现带来不同含义：

```
意象第一次出现：建立（观众注意到它）
意象第二次出现：变奏（同一意象，不同语境/景别/光线/状态）
意象第三次出现：完成（含义翻转或深化，观众突然理解）
```

**示例 — 一扇门：**
- 第一次：紧闭的，人站在外面 → 被排斥
- 第二次：半开的，光从缝里透出来 → 希望
- 第三次：打开的，人走进去 → 到达

在全片导演规划中，必须确定：
- **motif_item**：选什么意象（手/门/水/光/镜子/某个物件）
- **motif_arc**：三次出现分别在哪些 stage，每次的状态变化是什么

### 全片规划清单

先从完整 `stage_plan` 得出一条全片拍摄曲线，而不是一段一段孤立书写。

至少内部明确：

- `core_feeling`：这首歌最核心的感受是什么（一句具体的话，不是"伤感"）
- `visual_thesis`：整片最重要的视觉母题
- `motif_item` + `motif_arc`：1-2 个意象 + 每次出现的状态变化
- `energy_curve`：哪几段克制，哪几段释放
- `light_curve`：色温、对比度、主光方向如何推进
- `hero_stages`：哪些段必须出记忆点
- `breath_stages`：哪些段必须留呼吸
- `contrast_points`：哪些相邻段需要明显反差

### 大气连贯（Seedance 专属）

全片选 **1 种**主大气效果（薄雾 / 雨雾 / 粉尘 / 热浪 / 烟雾），写入 DIRECTING 的「大气」行，generator 在每个 shot 的 video_prompt 中带入，可微调浓度，不换种类。无明显大气需求则写「无特殊大气」。

### 廉价感警报

以下情况连续出现 2 次以上，基本会明显掉档：

- 连续中景正面表演，没有景别压缩和释放
- 每段都只是"人物看镜头唱/演"，没有阈值变化
- 光线很好听，但没有明确光源和主次
- 所有段都在"用力"，没有呼吸和留白
- 每段都把信息说满，观众没有补完空间

## 二、声画关系 + 音乐结构映射

### 声画关系（全片规划时确定，逐段可调）

| 类型 | 定义 | 效果 | 常见陷阱 |
|------|------|------|---------|
| **声画同步** | 画面直接配合音乐情绪和节奏 | 最直接，大部分商业 MV 的默认选择 | 上限低，容易变成"歌词字幕配图" |
| **声画分立** | 画面和歌词各自独立，在主题上共鸣而非直译 | 给观众留解读空间，情绪更深 | 做不好会让人觉得画面和歌不搭 |
| **声画对位** | 画面和音乐故意形成反差 | 最难但最有冲击力——反差本身在说话 | 过度使用会让人困惑 |

**推荐策略**（一支 MV 可混用三种）：
- **Verse** → 声画分立（建立情绪空间，不直译歌词）
- **Chorus** → 声画同步（情绪释放，画面跟着走）
- **Bridge** → 声画对位（最深的那句话，用反差说出来）

### 音乐结构映射表

| 结构 | 强度 | 光线策略 | 声画关系倾向 | 核心指令 |
|------|------|---------|------------|---------|
| Intro | 低 | 单光源留暗 | 分立 | 第一帧就要抓力，但不打满 |
| Verse | 低→中 | 克制侧光 | 分立 | 让人物和场景先成立；不要直译歌词 |
| Pre-chorus | 中→高 | 空间压缩 | 分立→同步 | 蓄力但不释放 |
| Chorus / Drop | 高 | 强对比 / 强色温 | 同步 | 至少 1 个截图即成立的画面 |
| Bridge | 转向 | 反差光线 | 对位 | 画面和音乐故意做反差——最深的那层意思 |
| Outro | 余震 | 松弛 | 分立 | 保留改变后的空气感；留白让观众完成 |

## 三、Treatment 写法

### Treatment 必须携带的信息

每条 treatment 必须包含以下维度，generator 将直接执行这些指令：

1. **视觉落点清单**：不同观察主体的「观众发现」，用 ①②③... 编号，每个落点开头必须加主体 tag。落点必须来自不同类别（人物动作/人物局部/物件/环境/光影），每个落点是 generator 的一个 beat 素材。主体 tag 规则：`[角色名]`（注册角色，generator 据此映射 @图片N）/ `[scene]`（场景）/ `[prop]`（物件）/ `[light]`（光影）
2. **身体状态关键词**：用英文括号标注 `(body: ...)`，generator 直接用于 beat 序列的角色状态描述。**写可拍的物理状态**（如 `jaw set, lips pressed, one slow swallow`），禁止情绪泛词（如 `sad, happy`）。眼神落点优先写轨迹而非状态（如 `gaze lifts from ground, stops mid-way, then snaps to camera` 而非 `eyes forward`）
3. **氛围**：光线方向 + 情绪底色
4. **→衔接**：本段结束如何引出下一段

> **stage 是片段，不是镜头（shot）。** planner 将 stage 组织成 SHOT（含合并/拆分），然后从输入画面描述中提炼视觉落点。每个落点对应 generator 的一个 beat 素材。如果只写一条连续动作，generator 只能把这个动作切成时间片段，导致每个 beat 都在看同一个人。
>
> **不指定焦段、景别、运镜**——那是 generator 的工作。planner 只说「观众看到什么」，不说「用什么镜头拍」。
>
> **音乐结构影响落点密度**：Chorus/高能段的落点可以比 Verse 更密、更局部（手指/眼神/局部特写），为 generator 的快切组提供素材；Verse/Intro 落点宜稀疏，给 generator 留空间做慢节奏 beat。

### 落点数量

planner 的落点是**素材单元**，generator 会将每个落点展开成 1-3 个 beat（身体传导、眼神轨迹、附属物延迟等）。planner 不控制展开密度，只控制落点数量。

按 shot 时长弹性调整：

| shot 时长 | planner 落点数 | 说明 |
|-----------|--------------|------|
| 4-5s | 2-3 | 每个落点 generator 展开约 1-2 个 beat |
| 6-8s | 3-4 | 每个落点 generator 展开约 2 个 beat |
| 9-12s | 4-5 | 留 1 个收束落点给 generator 自行设计 |
| 13-15s | 5-6 | 留 1-2 个收束落点给 generator 自行设计 |

**落点选择规则：**
- 6s 以上的 shot，落点中至少包含 1 个非人物类别（物件/环境/光影），为 generator 提供呼吸 beat 素材
- 避免所有落点都围绕同一人的不同动作——那只是"一个表演的分镜"，不是"多个发现"
- Chorus/高能段落点可更局部（手指/眼神/局部特写），generator 会据此触发快切组；Verse/Intro 落点宜稀疏宏观

### Hero Frame 标注

在 hero frame 所在 SHOT 的 treatment 末尾追加 `[hero]`。

### Motif 标注

在 motif 出现的 SHOT 末尾追加 `[motif:X·第N次]`，如 `[motif:门·第二次]`。

### 声画策略标注

需要特别声明声画关系时，在 SHOT 末尾追加 `[声画分立]` / `[声画对位]`。

### Treatment 写法示例

```
SHOT_1 [0-8s] (S1): ①[prop]手停在琴键上方悬而不落 ②[Kaiya]坐琴旁身体微弓不动(body: neutral expression, body containing something unsaid) ③[scene]纱帘街灯斜纹在地面缓慢移动 ④[light]琴键表面反射的街灯光斑随纱帘摆动而游走 | 克制·侧光压暗 →起身脚步引出下段

SHOT_4 [18-26s] (S4): ①[Kaiya]侧后方看他沿窗走过纱帘掠前景 ②[prop]手指沿窗框木纹划过停住(body: jaw set, lips pressed, shoulders held tight) ③[scene]纱帘投影在地面变形拉长 ④[light]窗外路灯在他肩上画出一道缓慢移动的光带 | 压抑·窗外逆光 →硬切回室内形成呼吸断裂 [hero]

SHOT_7 [38-43s] (S7): ①[scene]空房间只有门半开光线透入 ②[light]地面灰尘在光柱中缓慢上升 ③[scene]远处隐约脚步声但人未出现 | 等待·单光源 →对比前段的拥挤 [motif:门·第二次] [声画分立]

SHOT_10 [58-70s] (S10): ①[prop]指尖沿木纹划过停住像写完最后一笔 ②[prop]木纹年轮在微距下像指纹一样清晰 ③[Kaiya]抬头时眼睛湿但没有泪(body: heavy-lidded eyes, soft unfocused gaze) ④[Kaiya]手从木面缓慢抬起悬在半空指尖还在微颤 ⑤[scene]窗外天色从深蓝变为灰白 | 收束·散射柔光 →抬头动作接下段远景
```

## 四、蒙太奇策略 + 反差与反重复

### 蒙太奇思维

蒙太奇的本质：**A + B = C，而 C 是 A 和 B 各自都没有的意思。**

| 类型 | 做法 | 适用场景 | 在 treatment 中如何标注 |
|------|------|---------|----------------------|
| **节奏蒙太奇** | 剪辑节奏跟随音乐，张弛对比制造节奏感 | Chorus / Drop 的快切段 | treatment 内标注节奏密度 |
| **对比蒙太奇** | 相邻 shot 并排对立画面 | 需要情绪碰撞的 stage 交界处 | `→对比:` 标注对位关系 |
| **隐喻蒙太奇** | 插入自然界或物件画面暗示情感 | 呼吸镜头 / bridge 段 | treatment 中描述隐喻意象 |
| **积累蒙太奇** | 同一主题画面反复出现逐渐加强 | motif 成长弧线的三次出现 | 配合 motif_arc 使用 |

全片至少有 2-3 处主动运用蒙太奇策略构建"第三层意义"。

### 反差

全片至少 2 次明显反差跳跃：景别远↔近 / 动↔静 / 冷↔暖 / 封闭↔开阔

### 反重复

若相邻两段同时满足以下三项，基本等于无效重复：

1. 同一场景
2. 同一情绪段位
3. 同一构图和镜位关系

至少推进一项：新空间阈值 / 新物件状态 / 新身体姿态 / 新关系距离 / 新光线状态 / 新情绪重量

```
上一 shot 主要特征 → 本 shot 优先方向:
  正脸人像   → 背影 / 侧面 / 局部 / 空镜 / 反射
  固定镜头   → 运镜（推 / 拉 / 环绕 / 跟随）
  暖色调     → 冷色或中性
  近景       → 全景或特写（跳过中景）
  人物独占   → 加入环境 / 前景 / 物件
  高强度     → 呼吸 / 留白
```

## 五、Hero Frame

每支 MV 至少 2-3 个强定格画面，优先落在：开场第一记抓人画面、副歌开头、明确转折点、结尾余味段。

**Hero Frame 判断法**（至少满足其中 3 条）：

- 轮廓一眼能认
- 身体线条清楚
- 光源关系清楚
- 背景足够干净或足够有秩序
- 缩成缩略图也能成立
- 就算静音截图也有情绪

### 色调表

> 以下色调表和推荐相机仅在渲染族为摄影系时使用。画风系或参考图主导时，不推荐相机型号和胶片模拟。

### 光线词库

**人造光源：** `A single lamp` / `phone screen` / `colored storefront sign` / `bare overhead bulb`
**自然光源：** `Late afternoon sun through blinds` / `pre-dawn grey` / `overcast diffused` / `Golden hour sunlight through curtain gap`
**叙事功能光：** `Car headlight sweeping`（关系打断）/ `Doorway flooding with light`（阈值变化）/ `Phone screen spill`（私密夜晚情绪）

**光线叙事功能：** 高对比强侧光→冲突/决断 / 柔和漫射→脆弱/停顿 / 背光逆光→离开/转折/剪影

**衰减速度：** `rapid falloff`（切断）/ `gradual falloff`（柔和）/ `hard cutoff`（戏剧最强）

## 七、禁止词

```
dramatic / epic / breathtaking / stunning / gorgeous / beautiful / magical
cinematic lighting（必须写具体光源）
ethereal glow / glowing particles / floating orbs / magical particles
neon / 霓虹（用 colored gel light / colored LED strip / colored storefront sign 替代）
ambient warm light / warm light + cool light（无主次）
```

**运镜禁止（用三级精确术语替代）：** 缓慢移动 / 镜头变化 / 画面切换
**情感禁止（写可拍物理状态）：** sad expression / happy face
**物理矛盾禁止：** wide angle + shallow DoF / long shot + extreme bokeh / close-up + wide angle

## 八、硬性检查项

```
□ 全片先做过导演规划，不是逐段随手写
□ 内部确定了 core_feeling（具体感受，不是泛词）
□ 内部确定了 motif_item + motif_arc，并通过 [motif:X·第N次] 编码在 treatment 中
□ 内部确定了声画策略，并通过 [声画分立] 等标注编码在 treatment 中
□ 内部确定了全片主大气效果，写入 DIRECTING 大气行
□ 道具锁定已完成，写入 DIRECTING 道具锁定行
□ 所有 stage 都被覆盖，且顺序未被破坏
□ 音乐结构和镜头强度匹配（查「二、声画关系」映射表）
□ 至少 2 次明确反差跳跃
□ 至少 2-3 个 hero frame，通过 [hero] 标注
□ 至少 2-3 处主动运用了蒙太奇策略（对比/隐喻/积累）
□ 每个 treatment 落点开头带主体 tag（[角色名]/[scene]/[prop]/[light]）
□ 每个 treatment 落点数量与 shot 时长匹配（4-5s:2-3 / 6-8s:3-4 / 9-12s:4-5 / 13-15s:5-6）
□ 落点类别覆盖多样（人物动作/局部/物件/环境/光影），不是同一主体的不同角度
□ 每个 treatment 包含"→"后的衔接提示
□ 人物动作具体到身体部位，不是散文
□ 有角色的 SHOT 未被降级为空镜
□ 超长 stage（>15s）已拆成多个合法时长 SHOT
□ 超短 stage（<4s）已并入相邻 SHOT
□ 留白足够——不是每一秒都填满
□ SUMMARIZE 是 2-3 句 language_code 语言的视觉概述
□ DIRECTING 中渲染族已标注
□ 摄影系时 DIRECTING 包含 cam 推荐
```
