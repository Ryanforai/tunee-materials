# 古风 / 中国风 / 国风 · 视觉拍法参考

> 由 SKILL.md 识别到古风/中国风时加载。为 Preflight 推导 visual_identity 提供拍法锚点。

## 风格总纲

- **剪辑**：慢节奏为主，武打段一张一弛（慢动作亮相 + 快速动作 + 定格），大量空镜（山水/云雾/花瓣/月光）
- **机位**：缓慢环绕（展示服装飘逸感）；航拍/鸟瞰山水大场景；水墨长卷式缓慢横移
- **色彩**：淡雅水墨路线（低饱和/留白/青灰）或浓烈古典路线（高饱和红金/暖黄/高对比暗部）
- **标志手法**：大量留白（国画虚实）、框架构图（亭台门窗框景）、前景遮挡（竹叶/纱帘/花枝）
- **与西方 MV 关键差异**：空镜作为叙事主体（非过渡）；色彩作为结构性叙事工具；追求平面感/国画感而非景深立体感

## 子风格

### 水墨淡雅 / 文人
- **剪辑**：极慢（6-12s），空镜占比 40%+。转场用叠化/黑场
- **机位**：Static + Cinematic 固定留白构图 / Truck + Slow 缓慢横移长卷感
- **色彩**：低饱和，大量留白，青灰色调，接近水墨画
- **特征**：人物置于一侧大面积天空/山水，极简极静。古琴/箫段落对应静态或极缓镜头

### 浓烈古典 / 宫廷
- **剪辑**：中等节奏（4-6s），空镜占比 15-25%。转场用框景转场
- **机位**：Orbit + Smooth 环绕展示服装 / Push In + Cinematic 推进宫殿/庭院。居中对称构图受宫殿建筑影响
- **色彩**：高饱和红金，侧面硬光，暖黄底色，高对比暗部。张艺谋式色彩叙事（以色定情：红=权/欲，金=贵/暖，青=冷/净）
- **特征**：服装和场景的华丽感是视觉中心，构图饱满，前景遮挡增加层次。升格展示服装飘逸

### 仙侠 / 玄幻
- **剪辑**：中偏慢（4-7s），空镜占比 20-35%。转场用光效转场
- **机位**：Crane Up + Epic 升起展示仙境 / Dolly Zoom 变焦制造超现实感。垂直构图倾向（悬崖/瀑布/飞天轨迹）
- **色彩**：冷蓝 + 体积光 + 云雾为主，发光材质。暖金点缀（法力/仙气的发光效果常偏暖金）
- **特征**：体积光（丁达尔效应）大量使用，云雾作为常驻大气效果。升格展示飞天/施法

### 武侠 / 动作
- **剪辑**：一张一弛——亮相升格（4-7s）+ 交手快切（1-3s）+ 定格。转场用动作匹配剪辑。空镜占比 10-20%
- **机位**：Tracking + Fast 快速跟拍打斗 / Static + 突然 Zoom In 亮相变焦推近
- **色彩**：因场景而异——竹林青绿、沙漠土黄、夜战暗蓝
- **特征**：武器/招式特写，衣袂飘动升格，Speed Ramping（变速）是核心手法。"以慢写快"——极致升格细节（水珠碎裂/帽檐雨滴）反向表现速度感

### 国潮 / 新中式
- **剪辑**：中偏快（verse 3-5s，hook 2-3s），beat-sync 卡拍，跟现代说唱/电子节奏对齐
- **机位**：Handheld + Dynamic 手持动感 / Low Angle + Aggressive 仰拍气场 / Dutch Angle 倾斜
- **色彩**：红黑金 + 荧光蓝紫银，赛博朋克+古典混搭。高对比高饱和
- **特征**：解构主义拼贴——改制服装（非复原）、古典元素+现代场景混搭、glitch/色差特效。与传统古风的视觉语言完全不同

## BPM 与本风格的关系

古风音乐 BPM 跨度大（50-130），**剪辑节奏更多跟内容类型走而非 BPM**：
- 文人/水墨类 → 不管 BPM，都偏慢拍（6-12s），空镜为主
- 古典宫廷类 → 中等节奏（4-6s），展示华丽场景
- 仙侠类 → 中偏慢（4-7s），给体积光和特效呼吸空间
- 武侠类 → 动静交替——文戏 4-7s，武戏 1-3s 快切
- 国潮类 → 跟现代音乐节奏对齐（3-5s），不受传统古风节奏约束
