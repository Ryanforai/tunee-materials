# K-pop / 流行舞曲 · 视觉拍法参考

> 由 SKILL.md 识别到 K-pop 风格时加载。为 Preflight 推导 visual_identity 提供拍法锚点。

## 风格总纲

- **剪辑**：副歌极密集快切（1.2-2.5s），特写蒙太奇（眼→唇→手→饰品快速交替），主歌给表演空间（4-6s）
- **机位**：舞蹈段中景正面固定 + 多角度快切；转身/亮相瞬间 Orbit 环绕；对称构图配合队形
- **色彩**：高饱和高对比精细调色。每段可以有完全不同的色彩方案
- **标志手法**：特写蒙太奇（多部位快速交替）、对称构图、环绕亮相、预副歌景别急收（全→中→近→特写）+ 副歌瞬间爆发

## 子风格

### Girl Group / 甜酷路线
- **机位**：Orbit + Smooth 环绕展示造型 / Push In + Intimate 推近特写
- **色彩**：蓝粉霓虹 / lens flare / glitch 效果，高饱和糖果色
- **节奏**：预副歌景别急收，副歌瞬间爆发多角度。舞蹈 break 可用升格定格

### Boy Group / 力量路线
- **机位**：Low Angle + Dynamic 仰拍气场 / Tracking + Fast 快速跟拍舞蹈
- **色彩**：低饱和 + 高对比冷白高级感，或暗底金属质感
- **节奏**：舞蹈 break 用升格慢动作 + 定格，紧接快切回正常速

### Solo / 概念路线
- **机位**：Static + Cinematic 固定电影构图 / Dolly In + Dreamy 缓推梦幻
- **色彩**：一个核心色贯穿（粉/紫/蓝），明暗对比强
- **节奏**：概念段落可以放慢（5-7s），但副歌必须回到快切密度

## 与 EDM 的视觉借鉴

现代 K-pop 大量使用 EDM 的 build-up/drop 结构。视觉上体现为：
- 预副歌 = build-up：景别逐步收紧、剪辑加密、色彩渐强
- 副歌 = drop：瞬间爆发——场景切换、色彩方案突变、多角度密集切换

## BPM 与本风格的关系

K-pop BPM 通常 100-140，但**不管 BPM 多少，副歌段都需要密集分镜**。这是风格惯例，不是 BPM 驱动的：
- BPM 100 的抒情 K-pop → 主歌可以慢（5-6s），副歌仍要多角度切换（2-3s）
- BPM 130 的舞曲 → 全程高密度，副歌极速切（1.2-2s）
