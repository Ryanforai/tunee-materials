# R&B / Soul · 视觉拍法参考

## 风格总纲

- **剪辑**：中偏慢（verse 4-6s，chorus 3-5s），近景和特写占比高，柔和过渡多于硬切
- **机位**：近距离手持跟拍营造亲密感；缓推面部；升格捕捉肢体细节（手指/肩线/发丝）
- **色彩**：暖色调（琥珀/蜂蜜/暖棕），柔和低对比，烛光质感。或夜晚城市冷暖交织
- **标志手法**：极浅景深光斑、私密空间、皮肤质感特写、剪影构图（逆光剪影表达神秘和性感）

## 子风格

### Classic R&B / 暖色私密
- **机位**：Push In + Intimate 缓推近景 / Static + Dreamy 固定浅景深
- **色彩**：琥珀/蜂蜜/暖棕，烛光质感，柔和低对比
- **特征**：私密空间（卧室/沙发/车内），皮肤质感细节，极浅景深背景化为光斑

### Modern R&B / 城市夜景
- **机位**：Tracking + Handheld 手持跟拍城市漫步 / Dolly In + Cinematic 电影感推进
- **色彩**：冷暖交织——霓虹暖光（橙/粉）+ 冷蓝暗部，高对比。The Weeknd 式 neon-drenched 霓虹美学（红/紫/电子蓝）是 2020s 标志
- **特征**：夜晚城市作为情绪背景，反射面（雨水/玻璃/金属）放大光线层次。宽银幕画幅比增加电影感

### Neo-soul / 复古
- **机位**：Static + Cinematic 固定构图 / Slow Orbit + Smooth 缓慢环绕
- **色彩**：复古胶片色调（暖色偏移/颗粒感/柔和低对比），70年代质感
- **特征**：构图讲究，留白较多，节奏更从容（verse 5-7s）

### Alternative R&B / 实验
- **机位**：Handheld + Dreamy 手持梦幻 / Static + Cinematic 固定构图穿插超现实画面
- **色彩**：自然光为主，去饱和，穿插高饱和超现实片段
- **特征**：自然景观（海/森林/沙漠）融入情绪表达，超现实元素（物件漂浮/时间静止/空间扭曲）。Frank Ocean / SZA 式美学

## BPM 与本风格的关系

R&B BPM 通常 60-100，**整体偏慢拍，但 groove 感决定节奏而非纯 BPM**：
- BPM 65 的慢板 R&B → 长镜头（6-8s），升格慢动作，极其私密
- BPM 85 的中速 R&B → 中等节奏（4-6s），有律动但不急
- BPM 100 的 Uptempo R&B → 可以更快（3-4s），但保持柔和过渡，不用硬切
