# Rock / 摇滚 · 视觉拍法参考

> 由 SKILL.md 识别到 Rock 风格时加载。为 Preflight 推导 visual_identity 提供拍法锚点。

## 风格总纲

- **剪辑**：跟鼓点/吉他 riff 卡拍快切。**演奏画面和叙事画面交叉剪辑是摇滚 MV 最核心的结构**。高潮段骤然加速
- **机位**：手持为主（真实粗糙的晃动感）；演奏镜头多角度快切；航拍主要用于 Classic Rock 和 Post-Rock 展示空旷感
- **色彩**：暗调高对比去饱和，偏土色/冷灰/墨绿，胶片颗粒感。或直接黑白
- **标志手法**：乐器/演奏特写（琴弦/踏板/鼓面/汗珠）、手持晃动、演奏+叙事交叉剪辑

## 子风格

### Classic Rock / Hard Rock
- **剪辑**：中偏快（verse 3-5s，chorus 2-3s），高能量段可低至 1.5s，跟鼓点卡拍
- **机位**：Tracking + Handheld 手持跟拍 / Low Angle + Dynamic 仰拍舞台气场
- **色彩**：暖色暗调（琥珀/棕/红），高对比，胶片颗粒
- **特征**：演奏特写占比高，吉他手/鼓手特写交替。航拍展示大场景

### Indie / Alternative
- **剪辑**：中等节奏（verse 4-6s，chorus 3-4s），不追求极端冲击
- **机位**：Static + Cinematic 固定简约构图 / Handheld + Intimate 手持亲密感
- **色彩**：明亮自然色彩或复古胶片色调，可以比传统摇滚更清新
- **特征**：叙事性更强，日常场景入镜，不一定要演奏画面。内部跨度大：dream pop 朦胧柔焦 vs post-punk 冷硬锐利 vs lo-fi 粗糙手持

### Punk / 硬核
- **剪辑**：快切为主（verse 2-3s，chorus 1.5-2s），粗糙不精致
- **机位**：Handheld + Aggressive 手持晃动 / Push In + Fast 快速推进
- **色彩**：高对比黑白或去饱和冷调，VHS/监控摄像头质感
- **特征**：刻意的粗糙和不完美，画面可以过曝/欠曝/模糊。几乎不用航拍

### Metal / 重金属
- **剪辑**：极速切（chorus 1-2s），跟双踩/blast beat 卡拍
- **机位**：Low Angle + Aggressive 仰拍压迫感 / Orbit + Fast 快速环绕
- **色彩**：极暗调，几乎全黑 + 高光点缀（火焰/闪电/聚光灯）
- **特征**：演奏暴力美学——鼓槌特写、甩头慢动作、火焰/烟雾。升格穿插制造呼吸

### Post-Rock / Shoegaze
- **剪辑**：极慢（6-12s），接近 Ballad 节奏，延时摄影常见
- **机位**：Static + Dreamy 固定长镜头 / Crane Up + Slow 缓慢升起 / 航拍自然风景
- **色彩**：朦胧柔焦，低对比，光晕/过曝。或去饱和冷调
- **特征**：自然风景为主体，人物可以很小或不出现。与其他 Rock 子风格视觉差异巨大

## BPM 与本风格的关系

Rock BPM 跨度大（80-200+），**剪辑节奏主要跟鼓点密度和能量级别走**：
- BPM 90 的 Indie → 中等舒缓（4-6s），不需要快切
- BPM 120 的 Classic Rock → 中偏快（3-4s），跟鼓点卡拍
- BPM 160 的 Punk → 快切（2-3s），粗糙硬切
- BPM 200 的 Metal → 极速切（1-2s），但升格慢动作穿插制造呼吸
- BPM 130 的 Post-Rock → 极慢（6-12s），能量在持续音墙中而非鼓点
