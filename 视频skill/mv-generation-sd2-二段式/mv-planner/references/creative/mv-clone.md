# mv_clone · 参考重构型 · Seedance 2.0

> 输入已经给出每段的视觉意图（从参考 MV 映射而来）。本文件负责把这些参考镜头内容做成真正有层次的 Clone MV——不是照搬，是视觉 DNA 移植。

## 视觉 DNA 移植原则

- 每段先从 `md_stages` 的 `画面描述` 中提取参考镜头的视觉 DNA（景别/运镜/光线/主体动作/空间关系/物件交互/视觉落点）
- 只允许替换：参考角色编码 → 映射后的 `character_name`；参考场景编码 → 映射后的 `scene_name`
- 禁止重新诠释、抽象化、用自创内容替换参考镜头内容
- 同一参考镜头 DNA 在不同段落中复现时，必须改变至少一项：载体 / 景别 / 光线 / 颜色 / 空间关系

### DNA 元素提取

每段写 treatment 前，识别该行 `画面描述` 中的以下 DNA 元素：

| DNA 元素 | 提取内容 | 传递给 treatment 的方式 |
|----------|----------|----------------------|
| 景别 | 微距/特写/近景/中景/全景/大全景 | 影响落点选择 |
| 运镜 | 静态/推镜/拉镜/摇镜/手持/轨道等 | 影响运动节奏 |
| 光线 | 方向、色温、强度、落在哪个表面 | 写进落点的环境描述 |
| 主体动作 | 角色或环境在做什么 | 写进落点的动作描述 |
| 空间关系 | 角色与环境的相对位置 | 写进落点的空间锚点 |
| 物件交互 | 角色与道具的互动方式 | 写进落点的物件描述 |
| 视觉落点 | 最后一帧的定格画面 | 写进落点清单的最后一项 |

### 重复段落升级

当同一音乐段落（如 Chorus x2、Chorus x3）重复出现时，每次出现必须在至少一个维度上升级：

| 升级维度 | 示例 |
|----------|------|
| 景别升级 | 中景 → 近景 → 特写 |
| 光线升级 | 柔和 → 强烈 → 极端 |
| 距离升级 | 远 → 近 → 极近 |
| 情绪升级 | 克制 → 释放 → 爆发 |

视觉每次必须可区分。

## Clone 专属全片规划项

在全片导演规划基础上额外明确（directing.md §一的补充）：

- **`visual_dna`**：参考 MV 最重要的视觉 DNA（色调、光线质量、剪辑节奏），全片复用
- 每个 stage 有 **一个视觉落点主线**（参考镜头的核心视觉事件）
- 同一场景多次回访时，必须至少改变：光线状态 / 空间状态 / 大气浓度

---

## 场景回访

当同一个场景在片中多次出现时（Clone 模式常见），每次回访必须至少改变一项：

- 光线状态（方向/色温/强度）
- 空间状态（物件位移/痕迹变化）
- 大气浓度（薄雾加深/烟雾消散）

---

## Clone 专属反重复

（补充 directing.md §四的反重复规则）

- 同一参考镜头 DNA 不可无变化复用——必须改变构图、光线、色彩或空间关系
- 相邻 shot 不可同时满足：同场景 + 同情绪段位 + 同景别参考来源
- 同一场景的多次回访不可使用相同的光线签名

---

## 廉价复制警报

以下情况会让 Clone MV 掉到"低质翻拍"：

- 每个 shot 都机械照搬参考镜头，没有自己的视觉判断
- 全片没有呼吸——每个 shot 都在用力还原参考
- 角色与场景映射错误导致参考镜头内容水土不服
- 忽略了参考 MV 的剪辑节奏，只是静态还原单个镜头

---

## 专属禁止词

`copy` / `replica` / `exact same` / `identical`（Clone 是视觉 DNA 移植，不是像素复制）

`reinterpret` / `reimagine` / `creative liberty`（除非 stage_plan 明确要求创意改编）
