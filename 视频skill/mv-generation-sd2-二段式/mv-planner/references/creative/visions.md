# visions · 视觉概念型 · Seedance 2.0

> 输入已经给出每段的视觉意图。本文件定义如何把这些段落做成真正有层次的概念 MV——不是翻译剧本，是视觉诗人在作画。

## 概念表达原则

- 先从 `stage_plan` 提炼 1-2 个反复出现的核心意象（visual_motif）
- 同一意象重复出现时，必须改变至少一项：载体 / 景别 / 光线 / 颜色 / 空间关系
- 不要把同一个技巧连用三次；概念片需要变奏，不需要口号重复

### 概念嵌入方法

概念必须嵌入具体物质——不拍"孤独"，拍"空椅子上光线缓慢移动"；不拍"时间流逝"，拍"墙上影子角度变化"。

| 抽象概念 | ❌ 直接写 | ✅ 嵌入物质 |
|---------|----------|------------|
| 孤独 | "孤独的氛围" | 空旷房间中央一把椅子，侧光只照亮椅面 |
| 时间 | "时间流逝" | 墙上光斑从左移到右，影子角度渐变 |
| 记忆 | "回忆涌上" | 手指触碰旧照片边缘，照片表面反光微动 |
| 挣扎 | "内心挣扎" | 手握紧又松开，指甲在掌心留下印记 |
| 自由 | "自由感" | 风吹起薄纱帘直到填满整个画面 |

### 概念片的高级感来源

- 抽象概念必须落在具体载体上：皮肤 / 玻璃 / 水 / 金属 / 雾 / 布料 / 影子
- 一个意象先建立，再变奏，再升级，不要第一次出现就打到最满
- 真正厉害的 visions 不是"什么都怪"，而是"有一套稳定规则在慢慢失控"

## Visions 专属全片规划项

在全片导演规划基础上额外明确（directing.md §一的补充）：

- **`visual_motif`**：1-2 个反复出现的视觉元素（如：玻璃折射 / 水面倒影 / 门缝光线），贯穿全片
- **`concept_carrier`**：全片核心概念由什么物理材质承载（皮肤 / 玻璃 / 水面 / 金属 / 薄雾 / 织物 / 影子）
- 每个 stage 有 **一个情绪锚点** + **一个主视觉载体**（人/物/光/反射/大气）
- 同一画面不可无变化复用——必须改变构图、光线、色彩或空间关系

## 场景类型参考（非硬性路由，可混合）

### [写实]

克制运镜（Subtle/Static/gentle Dolly），精致写实用 Smooth 修饰，纪录写实用 Handheld。

最开始要足够像现实，异常才会有杀伤力。先从输入描述中建立真实基底，再让意象生长。

### [超现实]

三段升级：正常 → 轻微违和 → 明显异常。异常必须具体可拍（花从裂缝加速生长 / 水珠向上漂浮），不写"超现实的氛围"。

**异常升级梯度（至少命中 3 步）：**

1. 真实基底建立
2. 单一小异常（一个物理细节）
3. 异常换载体（物件 → 反射 / 水面 / 灯光）
4. 异常靠近人物（开始回应人物）
5. 阈值瞬间（异常突然停止 / 归位 / 爆出反差）

**异常载体词库：**

- **物件：** `a glass of water stays perfectly still during an earthquake` / `a clock with no hands still ticking`
- **反射：** `reflection shows a different time of day` / `mirror shows a gesture not being made`
- **影子：** `shadow facing the wrong direction` / `shadow slightly ahead of the person`
- **材质：** `fabric rippling in still air` / `stone wall breathing slowly`

### [奇幻]

先定世界规则（重力/光源/材质），全片一致，异质元素最多 1-2 种。

**异质化策略：**

- **比例失调：** `human figure dwarfed by architecture ten times their height`
- **物理违背：** `explosion frozen mid-blast, glass shards suspended mid-air`
- **非人感：** `skin becoming translucent, veins and bones visible beneath`

### [意识流]

四层光线签名：

```
[现实层] → 中性光 4500-5500K，对比度正常
[记忆层] → 过曝偏暖 2700-3500K，轻微虚焦，gradual falloff
[幻想层] → 欠曝偏冷 6500-9000K，色彩偏移，rapid falloff
[恐惧层] → 高对比极暗冷蓝青，hard cutoff
```

层切换需触发物特写（手持物 / 反射 / 身体部位 / 环境细节）。

## Visions 专属反重复

（补充 directing.md §四的反重复规则）

- 视觉载体不可连续 3 次相同（如不可 3 个 shot 都以"人物脸部"为载体）
- 同一光线签名不可连续使用
- 概念表达必须层层递进或螺旋回扣——不是 3 次平行重复同一概念

---

## 廉价梦境警报

以下组合很容易让 visions 掉到"AI 梦境拼贴"：

- 每段都极端，没有基线
- 每段都只靠发光、漂浮、烟雾
- 没有稳定主色，只是一堆好看颜色
- 每段都在说"神秘"，但没有具体载体

---

## 专属禁止词

`surreal` / `dreamlike` / `otherworldly` / `uncanny`（必须写具体异常的物理形态）

`linear narrative` / `clear cause and effect` / `single timeline`（除非 stage_plan 明确要求叙事结构）

### Case：废弃温室 · 超现实概念片

> visions · 超现实 | 色调: 冷灰蓝→暖琥珀 | 情绪: 消逝与重生
> **参考价值：** 写实基底建立后异常升级、视觉载体交替（材质→光→人物→奇观）、概念嵌入物质

**summarize**：以废弃温室为舞台的视觉概念片。冷灰蓝渐变暖琥珀，薄雾贯穿。从锈铁与枯叶的静默对话到镜面碎裂中的自我重组，Yuki 完成一场关于重生的视觉仪式。

**SHOT_2 [8-15s] (S2)：** ①[prop]锈蚀铁架表面铁锈纹理像干涸的河床，一滴冷凝水从上方滑落击中锈面溅开微小水花 ②[light]水珠中折射出穹顶破洞外的天空——一小块蓝白 ③[Yuki]从光柱边缘缓缓走入光柱，灰尘在光柱中因她的经过旋转上升(body: eyes upward, lips parted, weight shifting into the light) ④[scene]穹顶破碎玻璃的缝隙中光束斜切而下如固体柱子 | 冷灰蓝·写实基底·孤独感 →手触枯藤引出下段异常 [motif:水·第一次]

**SHOT_3 [15-21s] (S3)：** ①[Yuki]手指抚过一根枯死的藤蔓，指腹触碰处干皮碎裂簌簌坠落(body: fingers barely touching, breath held) ②[prop]指尖下露出藤芯内部一线尚存的翠绿——异常：还活着 ③[light]这条翠绿发出极微弱的自发光，在冷灰蓝基底中像一枚针刺 | 冷灰蓝·异常初现·生命信号 →光持续变强引出奇观段 [hero] [motif:绿·第一次]
