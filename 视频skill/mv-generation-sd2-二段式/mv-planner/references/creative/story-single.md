# story_mode · 单线 · Seedance 2.0

> 输入已经给出单线剧情的 stage 规划。本文件负责把这条线拍得更有戏、更有层次、更像导演作品。

## 风格判断

从完整 `stage_plan` 的人物关系、空间气质、音乐结构推断拍摄风格。`visual_style` / `effective_style` 存在时优先。

### 精致写实（默认）

- 运动：缓慢推进 / 固定静观 / 轻柔跟随 / 精确滑轨 / 缓慢环绕
- 禁止：无理由手持晃动 / 急速运镜 / 为炫技而炫技
- 景别：丰富变化，但每次变化都服务叙事推进
- 景深：浅景深优先，长焦（85–135mm）压缩空间
- 光线：精心单侧布光，完整三层光（主光 + 衰减 + 补光）
- 人物：克制精准，每个动作都带叙事含义

### 纪录写实

- 运动：手持晃动 / 跟拍 / 固定窥视 / 偷拍感长焦
- 禁止：过于顺滑的稳定器语言 / 甜美广告式构图
- 景别：中景/近景为主，允许不完整构图
- 光线：自然光 / 环境光，允许局部不均匀

## 单线拍法

每个 stage 至少明确：

- 这段的不可逆动作是什么
- 这段的情绪增量是什么
- 这段最强的 hero frame 是什么
- 这段与上一段相比新增了什么

### 单线高级化方法

- 不要把"发生了什么"平着拍完，优先找出这段**最贵的一帧**
- 一个 stage 里只挑**一个最值钱的动作**，不要什么都拍
- 如果是情绪段，不要只拍脸；给空间、手部、物件、门槛

### 单线递进规则

- **开场段**：先给视觉悬念，不要立刻把全部信息说完
- **积累段**：用人物动作、空间阈值、物件状态推动，不靠抽象内心独白
- **转折段**：必须通过可拍到的动作完成（打开纸袋 / 拿起钥匙 / 推门进入 / 把外套搭上肩）
- **余波段**：让空间和身体状态保留变化后的后劲

### 同一场景内的推进

同一场景连续出现不等于重复。必须至少推进一项：

- 人物距离镜头更近或更远
- 人物从空间边缘进入中心，或反过来
- 物件状态发生变化
- 光线方向或强度变化
- 前景遮挡或景深层次变化

让同一场景拍出高级感：开始拍"人被空间吞住" → 中段拍"人和空间发生接触" → 结束拍"人改变了空间里的某个状态"

## 高级故事镜头的常见来源

- **阈值**：门框、楼梯、车门、窗前、走廊转角
- **前景**：玻璃、水汽、帘子、肩膀、门框边缘
- **关系距离**：同框但不接触 / 接触但不对视 / 对视但不靠近
- **物件状态**：钥匙、手机、衣领、杯子、镜子、信封
- **材质**：湿地面、旧木头、金属反光、布料褶皱

这些东西不是装饰，是把普通剧情拍贵的方式。

## Story 专属——表演模式

每段想清楚角色的表演模式：

- **对镜**：直视镜头，打破第四面墙，建立观众连接
- **回避**：背对、侧对、视线游移，内心世界不对外开放
- **抽离**：人在场但精神不在，凝视空处
- **静止**：情绪压制到极点的身体冻结
- **失控**：抑制不住的情绪溢出，身体先于意识反应

每段找到"最贵的一帧"——如果只能截一张图，那张是什么？

### 身体姿态优先级

- 下巴抬起但肩膀压低
- 一手抓住衣领或包带
- 背靠空间边缘，重心偏到一侧
- 身体已经转走，视线还没离开
- 手停在物件上方而不是已经拿完

### 表情与身体微状态（给 body: 标注用）

```
强势：chin raised, jaw set, heavy-lidded unyielding
忧郁：downcast eyes, soft unfocused, lips parted, brow furrowed
颓废：heavy-lidded, slow gaze, lips swollen, skin flushed
冷静：neutral, observing without engaging
脆弱：eyes wet, brows knitted, mouth trembling
神秘：slight closed-lip smile, gaze oblique
```

### Case：胶片复古暖调 · 海边恋爱短片

> story_mode · 单线 | 色调: 胶片复古暖调 | Acoustic pop / Ballad
> **参考价值：** 环境先于人物出场、自然元素（风/水/鸟群）制造画面运动、近景人像与远景海天交替

**summarize**：海边落日恋爱短片。前段用车窗、湖面、长椅、树影建立空间，女主延迟出场。中段用牵手奔跑、海浪嬉闹、沙滩自拍三组动作完成情侣关系。全片画面运动由风吹头发、海浪打脚、鸟群飞过承担，不需要复杂表演。结尾不用正脸，只留脚印、背影。

**SHOT_1 [0-8s] (S1)：** ①[scene]车窗上雨滴从上往下滑，积成一条弯曲的细流 ②[light]车窗外灰蓝天空倒映在雨滴内部 ③[scene]车内后视镜角落一束干燥的满天星随颠簸轻摆 | 公路上雨天·等待感 →车停，车门声引出人物

**SHOT_3 [18-26s] (S3)：** ①[Ren]站在长椅扶手上单脚保持平衡，手臂微张，风吹起外套下摆(body: chin raised, slight closed-lip smile, focused on balance) ②[scene]长椅下方积水映出她的模糊倒影随风涟漪 ③[light]夕阳从侧面将她的轮廓烧成金边 | 黄金时刻·恣意感 →跳下长椅接下段 [hero]
