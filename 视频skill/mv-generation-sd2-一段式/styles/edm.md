# EDM / 电子 · 视觉拍法参考

> 由 SKILL.md 识别到 EDM 风格时加载。为 Preflight 推导 visual_identity 提供拍法锚点。

## 风格总纲

- **剪辑**：极度卡拍——build-up 递增频率，drop 瞬间视觉爆发。闪频/闪白/快速叠化制造迷幻感
- **机位**：环绕/旋转配合律动；航拍大场景；微距特写快切（水滴/光线/粒子）；速度突变（正常→升格）
- **色彩**：霓虹色（品红/电蓝/亮紫/酸绿），暗底亮光——黑色背景中的激光/LED
- **标志手法**：drop 瞬间视觉爆发（暗→亮 + 饱和度拉满 + 镜头从静→动 + 画面从单层→多层叠加）、环绕律动、对称/万花筒效果

## 子风格

### House / Progressive
- **剪辑**：build-up 4-6s 渐收紧，drop 2-3s 快切但不极端
- **机位**：Orbit + Smooth 环绕律动 / Crane Up + Epic 升起展示场景
- **色彩**：暖色霓虹（橙/金/粉），比 Techno 更温暖
- **特征**：groove 感优先，不追求极端冲击，画面可以有"呼吸"

### Tropical / Melodic House
- **剪辑**：中等节奏（verse 4-6s，drop 3-4s），比 Club House 更舒展
- **机位**：Crane Up + Cinematic 航拍自然景观 / Tracking + Smooth 跟拍
- **色彩**：温暖日落色调（金/橙/粉），自然光为主，沙滩/海洋/棕榈意象
- **特征**：与 Club House 视觉差异巨大——更接近旅行 Vlog 的温暖质感而非夜店暗光

### Techno / 硬核电子
- **剪辑**：极高密度（drop 1-2s），闪频/strobe 效果
- **机位**：Static + Aggressive 固定但画面内容高速变化 / Dutch Angle 倾斜 + 闪切
- **色彩**：极简冷感——全黑 + 银色金属点缀 + 单色光源。Berlin 美学：原始工业空间、混凝土、钢铁
- **特征**：重复性和催眠感，同一画面微变化快速循环

### Dubstep / Bass Music
- **剪辑**：跟 bass drop 卡拍——drop 前静止/慢速，drop 瞬间极速切换
- **机位**：Push In + Aggressive 快推冲击 / Crane Down + Fast 急速俯冲
- **色彩**：暗底 + 闪烁高饱和色，每次 drop 可以换色
- **特征**：动静对比极端——安静段几乎不动，bass drop 瞬间画面炸裂

### Ambient / Chill Electronic
- **剪辑**：缓慢（5-8s），接近 Ballad 的节奏但更抽象
- **机位**：Slow Dolly In + Dreamy 梦幻缓推 / Static 固定长镜头
- **色彩**：柔和霓虹（淡紫/淡蓝/薄荷），低对比
- **特征**：光线和粒子可以做视觉主体，人物不必是画面中心

## BPM 与本风格的关系

EDM 的 BPM 通常 120-150+，但**剪辑节奏跟歌曲结构（build-up / drop）的关系远大于 BPM**：
- BPM 128 的 House → 整体中速流畅（3-5s），drop 略加速
- BPM 128 的 Tropical House → 更舒展（4-6s），drop 不追求极端冲击
- BPM 140 的 Trance → build-up 渐快，drop 密集但有延伸感（宇宙/星空/euphoric 光线）
- BPM 140 的 Dubstep（感知 ~70）→ 不看 BPM，看 bass drop 位置——drop 前静止，drop 后炸裂
- BPM 90 的 Ambient → 完全不快切，5-8s 长镜头
