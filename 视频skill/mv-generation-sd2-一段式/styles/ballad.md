# Ballad / 抒情 · 视觉拍法参考

## 风格总纲

- **剪辑**：慢切为主（verse 6-10s，chorus 4-6s），大量淡入淡出做段落转换，空镜头占比高
- **机位**：缓慢推镜是核心（Slow Push In 制造代入感和情绪积累）；固定机位或极缓横移；极少快速运镜
- **色彩**：整体倾向去饱和/柔和。暖色（琥珀/金/暖黄）或冷调忧郁（蓝灰/青）。全片统一，不频繁换色。黑白/单色处理也是经典选择
- **标志手法**：缓推特写、逆光剪影、大量留白构图、升格慢动作

## 子风格

### 暖色抒情 / 回忆
- **机位**：Slow Push In + Intimate 缓推面部 / Static + Dreamy 固定浅景深
- **色彩**：琥珀/金色/暖黄，柔和低对比胶片质感
- **特征**：景别缓慢递进（全→中→近→特写），用空间感制造情绪积累

### 冷调忧郁 / 离别
- **机位**：Pull Out + Slow 缓慢拉远（疏离感）/ Static 固定机位长镜头
- **色彩**：蓝灰/青色/去饱和，低对比。或直接黑白处理
- **特征**：大量留白，人物占比小，框架构图（门窗中的人影）。极少剪辑甚至 one-shot 长镜头也可行

### 大气史诗 / 情感爆发
- **机位**：Crane Up + Epic 升起（副歌升华）/ Orbit + Cinematic 环绕
- **色彩**：主歌冷调 → 副歌暖调渐变（情绪释放的色彩弧线）
- **特征**：副歌可以略加快（4-5s），但仍远慢于流行曲。用升格放大高潮瞬间

### Power Ballad / 摇滚抒情
- **机位**：Static + Cinematic 开场 → Tracking + Dynamic 乐队演出段落 / Crane Up + Epic 副歌升华
- **色彩**：暗调暖色为主，副歌光线增强（舞台灯光/逆光剪影）
- **特征**：乐队演奏画面 + 叙事画面交叉剪辑。verse 偏叙事（6-8s），chorus 可加速到 3-4s 配合乐队能量

## BPM 与本风格的关系

Ballad BPM 通常 60-85，**这是少数 BPM 和剪辑速度正相关的风格**——慢歌就该慢拍：
- BPM 60 极慢抒情 → 8-12s 长镜头，几乎不切
- BPM 75 中速抒情 → 6-8s 主歌，4-6s 副歌
- BPM 85 偏流行的抒情 → 可以稍快（5-7s 主歌，3-5s 副歌），但仍保持舒缓感
