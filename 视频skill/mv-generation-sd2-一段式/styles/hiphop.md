# Hip-hop / 说唱 · 视觉拍法参考

> 由 SKILL.md 识别到 Hip-hop 风格时加载。为 Preflight 推导 visual_identity 提供拍法锚点。

## 风格总纲

- **剪辑**：跳切硬切为主，很少柔和转场。可用分屏和闪频做风格化处理
- **机位**：低角度仰拍是标志性"权力镜头"；广角/鱼眼怼脸；手持跟拍带纪录片感
- **色彩**：跨度极大——阳光金色到暗黑冷蓝都有，完全取决于子风格
- **标志手法**：低角度权力镜头、车内 POV、人物居中占满画面、分屏对比

## 子风格

### West Coast / Chill
- **剪辑**：中速舒展（verse 4-6s，hook 3-4s），律动感优先于冲击感
- **机位**：Slow Dolly In + Intimate 近距离跟拍 / Low Angle + Dynamic 仰拍带出气场
- **色彩**：金色暖调，自然光，黄金时刻。主歌暖黄，副歌可加冷蓝对比
- **关键词**：慵懒、阳光、街头、敞篷车、棕榈树

### Trap / 暗黑
- **剪辑**：**BPM 低但切不慢**——verse 3-5s 跟 808 慢脉搏，hook 2-3s 跟 hi-hat 三连音密度
- **机位**：鱼眼/广角怼脸 + Aggressive / Dutch Angle + Aggressive 倾斜压迫 / Extreme Close-up + Handheld 手持
- **色彩**：暗底高对比，深蓝灰/暗红/品红。青橙分调（teal-orange）是当代最流行方案。hook 可用闪频效果
- **关键词**：808、hi-hat、暗黑、攻击性、低沉
- **注意**：Trap BPM 标记通常 130-160，但感知节拍为半速 65-80。prompt 中描述的节奏感应对齐感知节拍

### Old School / Boom Bap
- **剪辑**：中等节奏（verse 4-5s，hook 3-4s），跳切硬切为主
- **机位**：中景正面 + 多角度快切 / Low Angle 仰拍 / Tracking + Handheld 手持跟拍街头
- **色彩**：去饱和暗调或 VHS 复古颗粒感
- **关键词**：采样、鼓机、街头、地下

### Drill
- **剪辑**：中偏快（verse 3-4s，hook 2-3s），硬切配合 sliding 808
- **机位**：鱼眼广角 + Aggressive / POV + Handheld 第一人称手持 / Static + 人物逼近镜头
- **色彩**：因地域变体差异大——
  - UK Drill：冷灰去饱和 + 面罩/巴拉克拉瓦是核心视觉标识
  - Chicago Drill：粗粝低成本纪录片质感，街头实景
  - NY Drill：更多时尚元素，闪光灯白色爆闪
- **关键词**：街头、夜晚、闪光灯、直视镜头

## BPM 与本风格的关系

Hip-hop BPM 跨度大（60-110 感知），**绝不能只看 BPM 判断剪辑速度**：
- BPM 70（感知）的 Trap → 快切（hi-hat 三连音密度决定节奏感）
- BPM 95 的 West Coast → 中速舒展（放克贝斯律动决定节奏感）
- BPM 85 的 Boom Bap → 中等硬切（鼓点密度决定节奏感）
- BPM 75（感知）的 Drill → 中偏快（sliding 808 + 密集 flow 决定节奏感）
