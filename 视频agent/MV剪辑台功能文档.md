# MV 剪辑台 — 功能文档

> 基于 `mv-editor.html`（完整版：多轨道 + 歌词）和 `mv-editor-simple.html`（精简版：对话式 AI 剪辑 + 单视频轨+单音频轨）原型，记录所有交互功能。

---

## 一、顶部工具栏

### 1. 品牌标识
- **描述**：显示 Tunee Logo + 「MV 工作台」标题 + 「视频剪辑器 · 自动保存中」副标题
- **触发**：静态展示
- **实现**：`.topbar-brand` — `.topbar-logo`（圆形 "T"）+ `.topbar-info`（标题/副标题）
- **状态**：已完成

### 2. 浏览器风格标签页
- **描述**：显示当前编辑器标签「视频剪辑器」，带视频图标
- **触发**：静态展示
- **实现**：`.topbar-tab` — 圆角顶部标签样式，`border-bottom: none` 覆盖底部边框
- **状态**：已完成

### 3. 导出成片
- **描述**：导出最终 MV 视频文件
- **触发**：点击右上角「导出成片」按钮
- **实现**：`exportProject()` — 当前显示 toast 提示"导出功能开发中…"
- **状态**：占位（TODO）

### 4. 用户头像
- **描述**：右上角显示用户头像（圆形，首字母 "Y"）
- **触发**：静态展示
- **实现**：`.topbar-avatar` — 橙色圆形，hover 降低透明度
- **状态**：已完成

## 二、AI 对话面板（精简版独有，左侧 360px）

### 5. AI 助手对话
- **描述**：左侧面板展示与 Tunee AI 的对话，包含 AI 消息和用户消息气泡
- **触发**：页面加载时预填充 Demo 对话
- **实现**：`.chat-messages` — `.chat-msg.user`（右对齐，深色气泡）/ `.chat-msg`（左对齐，透明气泡）
- **状态**：已完成（Demo 数据）

### 6. 富内容卡片
- **描述**：AI 消息内可嵌入故事板卡片（可展开/折叠）、选项列表等富内容
- **触发**：页面加载时预填充
- **实现**：`.chat-card`（故事板卡片，header + 展开箭头）/ `.chat-options-card`（多选项列表 + 选中状态）
- **状态**：已完成（Demo 数据）

### 7. 选项选择
- **描述**：用户可在 AI 回复的选项卡片中选择要调整的镜头
- **触发**：点击 `.chat-option-item`
- **实现**：`selectOption(el)` — 切换 `.selected` class，橙色勾选标记
- **状态**：已完成

### 8. 文本输入 & 发送
- **描述**：底部文本输入框，支持多行输入，发送按钮
- **触发**：输入文字 → 点击发送按钮 或 Enter
- **实现**：`sendChat()` — 创建用户消息 DOM → 插入 AI "思考中" loading → 2s 后替换为模拟回复
- **状态**：已完成（模拟 LLM 调用）

### 9. 附件 & 工具按钮
- **描述**：输入框左侧附件（回形针）和工具（齿轮）图标按钮
- **触发**：点击
- **实现**：`.chat-icon-btn` — 静态图标按钮，hover 背景高亮
- **状态**：已完成（UI 占位）

### 10. 型号标签
- **描述**：输入框内显示 `Tunee MV · Pro` 标签（橙色点前缀）
- **触发**：静态展示
- **实现**：`.chat-tag` + `.chat-tag-dot`
- **状态**：已完成

## 三、素材面板（右侧编辑区左侧 30%）

### 11. 素材分组切换
- **描述**：按 All / A / B / C 四组切换素材集
- **触发**：点击顶部 `All` `A` `B` `C` 标签
- **实现**：`switchPanel(btn, panel)` — 更新 `S.panel`，重新渲染素材网格
- **状态**：已完成

### 12. 素材类型切换
- **描述**：按 视频 / 图片 / 音频 筛选当前分组素材
- **触发**：点击 `视频` `图片` `音频` 标签
- **实现**：`switchType(btn, type)` — 更新 `S.type`，重新渲染素材网格
- **状态**：已完成

### 13. 导入素材
- **描述**：从电脑本地选择文件导入素材库，自动识别类型（video/image/audio），添加到当前分组
- **触发**：点击素材网格末尾的「导入素材」卡片（虚线边框）
- **实现**：`importFile()` — 创建 `<input type="file" multiple>`，遍历文件推入 `MEDIA[S.panel][type]`，去重后重新渲染
- **状态**：已完成

### 14. 素材拖拽到轨道
- **描述**：将素材从素材库拖入任意轨道，插入模式（非覆盖），自动按时间排序
- **触发**：拖拽素材卡片 → 释放到轨道行
- **实现**：HTML5 Drag-and-Drop API
  - `dragstart`：序列化 `{name, dur}` 到 `dataTransfer`
  - `dragover`：绘制橙色插入指示线 `drawDropIndicator()`
  - `drop`：计算落点时间 → 找到插入位置 → `splice` 插入 → `pushHistory()` → `renderAllClips()`
- **状态**：已完成

## 四、视频预览区（右上）

### 15. 播放 / 暂停
- **描述**：控制播放状态，播放时按钮切换为暂停图标
- **触发**：点击播放按钮 或 视频画面 hover 播放覆盖层
- **实现**：`togglePlay()` — `setInterval` 以 100ms 步进更新 `S.t`，到达 `S.dur` 后自动归零
- **状态**：已完成

### 16. 进度条跳转
- **描述**：点击进度条任意位置，播放头跳转到对应时间
- **触发**：点击进度条区域
- **实现**：`seekFromBar(e)` — 计算点击位置百分比 → `S.t = p * S.dur` → 同步更新播放头 / 进度条 / 时间码 / 预览
- **状态**：已完成

### 17. 音量控制
- **描述**：弹出音量滑块，初始值 50%，范围 0-100%
- **触发**：点击音量图标按钮
- **实现**：`openVol(e)` / `closeVol()` / `setVol(v)` — Popover 定位在按钮上方，range slider 驱动 `S.volume`
- **状态**：已完成

### 18. 倍速控制
- **描述**：弹出倍速选择器，可选 0.8x / 1x / 1.2x / 2x，初始 1x
- **触发**：点击 `1×` 按钮
- **实现**：`openSpd(e)` / `closeSpd()` / `setSpeed(spd)` — Radio 列表，选中项高亮橙色，更新 `S.speed` 影响播放速度
- **状态**：已完成

### 19. 全屏
- **描述**：进入/退出浏览器全屏
- **触发**：点击全屏图标按钮
- **实现**：`goFull()` — Fullscreen API：`document.documentElement.requestFullscreen()` / `document.exitFullscreen()`
- **状态**：已完成

### 20. 视频预览三级降级
- **描述**：根据素材生成状态，预览区显示不同内容
- **触发**：自动（点击选中 clip 时 / 播放过程中实时更新）
- **实现**：`updatePreview(label, opts, time)`
  - **Level 1 — 视频已生成** (`videoReady: true`)：显示渐变背景模拟视频画面 + 右上角「视频」badge（橙色）
  - **Level 2 — 分镜图** (`videoReady: false, imageSeed: number`)：全屏渐变模拟分镜图（亮度 0.30+）+ 右上角「分镜图」badge
  - **Level 3 — 文字描述** (`videoReady: false, 无 imageSeed`)：居中白色文字，显示音乐结构标签 + 画面描述 + 底部"视频生成中"提示 + 右上角「描述」badge
- **状态**：已完成

### 21. 视频画面 hover 播放覆盖层
- **描述**：鼠标悬停在视频画面上时，显示半透明播放按钮覆盖层
- **触发**：hover 视频画面
- **实现**：CSS `:hover` 控制 `.video-hover-play` 的 opacity，点击调用 `togglePlay()`
- **状态**：已完成

### 22. 时间码显示
- **描述**：显示当前播放时间 / 总时长，如 `00:26 / 03:45`
- **触发**：自动更新
- **实现**：`updateTime()` — `fmtT(S.t)` 格式化，每 100ms 刷新，播放时实时更新当前片段预览
- **状态**：已完成

## 五、时间轴工具栏（底部上方）

### 23. 删除
- **描述**：删除当前选中的 clip
- **触发**：点击「删除」按钮 或 键盘 `Delete / Backspace`
- **实现**：`deleteSelected()` — 遍历 `CLIPS` 找到对应数据 splice 移除 → DOM remove → 清除 `S.sel` → `pushHistory()`
- **状态**：已完成

### 24. 分割
- **描述**：在播放头位置将视频轨中跨越播放头的 clip 一分为二
- **触发**：点击「分割」按钮 或 键盘 `S`
- **实现**：`splitAtPlayhead()` — 遍历视频轨 clip，找到包含 `S.t` 的 clip，拆分为两个新 clip → `pushHistory()`
- **状态**：已完成

### 25. 撤销
- **描述**：撤销上一次编辑操作（删除/分割/拖拽/导入）
- **触发**：点击「撤销」按钮 或 键盘 `Cmd/Ctrl + Z`
- **实现**：`undo()` — 从 `HISTORY.undo` 弹出快照恢复 `CLIPS`，推入 redo 栈 → `renderAllClips()` → 更新按钮状态
- **状态**：已完成

### 26. 重做
- **描述**：重做被撤销的编辑操作
- **触发**：点击「重做」按钮 或 键盘 `Cmd/Ctrl + Shift + Z`
- **实现**：`redo()` — 从 `HISTORY.redo` 弹出快照恢复 `CLIPS`，推入 undo 栈 → `renderAllClips()` → 更新按钮状态
- **状态**：已完成

### 27. 新增歌词字幕
- **描述**：AI 驱动，点击后立即显示歌词轨和 loading 状态，3 秒后填充歌词内容
- **触发**：点击「＋ 新增歌词字幕」按钮
- **实现**：`generateLyrics()` — 仅 `mv-editor.html` 完整版
- **状态**：已完成（模拟 LLM 调用，仅完整版）

## 七、Clip 交互

### 32. Clip 选中
- **描述**：点击 clip 选中，显示橙色边框
- **触发**：鼠标点击 clip（非 handle 区域）
- **实现**：`onClipDown()` → `selectClip(el)` — 清除其他选中 → 添加 `.sel` class → 更新视频预览
- **状态**：已完成

### 33. Clip 拖动
- **描述**：拖拽 clip 在轨道内移动，松开后按新位置排序并紧密排列
- **触发**：鼠标按下 clip 并拖动
- **实现**：
  - 拖动中：`onDragMove()` type `move` — 计算 `dx` → 更新 `left`
  - 松开后：计算新起始时间（0.5s 吸附）→ 排序 → 紧靠（每个 clip 起始 = 前一个 clip 结束）→ `pushHistory()`
- **状态**：已完成

### 34. 初始自动紧靠
- **描述**：打开页面时所有轨道的 clip 自动紧密排列，无间隙
- **触发**：页面加载 `init()`
- **实现**：遍历所有轨道，按 start 排序后依次重算起始位置
- **状态**：已完成

### 35. Clip 缩放（调整时长）
- **描述**：拖拽 clip 左右两端的 resize handle，调整 clip 的起始时间和时长
- **触发**：鼠标按下 clip 左侧/右侧 handle 并拖动
- **实现**：`onDragMove()` type `resize` — 左侧 handle：同时改 `left` 和 `width`；右侧 handle：只改 `width`，最小宽度 20px
- **状态**：已完成

### 36. 磁性吸附（已替换）
- **描述**：已改为拖拽结束后的插入排序 + 紧密排列，不再使用运行时磁性吸附
- **状态**：已由 #34 替代

### 37. 点击空白取消选中
- **描述**：点击时间轴空白区域取消当前 clip 选中
- **触发**：点击时间轴非 clip 区域
- **实现**：全局 `click` 监听 — 如果 `!e.target.closest('.clip')` 且在 `.tl-scroll` 内，清除 `.sel` 和 `S.sel`
- **状态**：已完成

## 八、播放系统

### 38. 播放头实时更新
- **描述**：播放时橙色播放头随时间移动
- **触发**：播放中每 100ms
- **实现**：`updatePH()` — `playhead.style.left = S.t * S.zoom + 'px'`
- **状态**：已完成

### 39. 播放头自动滚动跟随
- **描述**：播放头移出可视区域时，时间轴自动滚动跟随
- **触发**：播放中每 100ms
- **实现**：`togglePlay()` 内 — 当 `phPx > scrollLeft + clientWidth - 80` 时设置 `scrollLeft = phPx - sc.clientWidth / 2`
- **状态**：已完成

### 40. 倍速影响播放
- **描述**：选择的倍速影响播放步进速度
- **触发**：`setSpeed(spd)` 后播放时自动生效
- **实现**：`S.t += 0.1 * S.speed` — 倍速乘子直接作用于时间步进
- **状态**：已完成

### 41. 播放循环
- **描述**：播放到结尾后自动归零重新开始
- **触发**：`S.t >= S.dur` 时自动
- **实现**：`togglePlay()` 内 — 归零 `S.t = 0`，停止 timer，重置按钮图标
- **状态**：已完成

## 九、键盘快捷键

| 快捷键 | 功能 | 实现 |
|--------|------|------|
| `Space` | 播放 / 暂停 | `e.code === 'Space'` → `togglePlay()` |
| `Delete / Backspace` | 删除选中 clip | `e.code === 'Delete/Backspace'` → `deleteSelected()` |
| `S` | 在播放头处分割 | `e.code === 'KeyS'` → `splitAtPlayhead()` |
| `Cmd/Ctrl + Z` | 撤销 | `e.code === 'KeyZ'` → `undo()` |
| `Cmd/Ctrl + Shift + Z` | 重做 | `e.code === 'KeyZ' + shiftKey` → `redo()` |
| `Ctrl/Cmd + 滚轮` | 时间轴缩放 | `wheel` 事件 → 更新 `S.zoom` |
| `Enter`（chat 输入框中） | 发送消息 | `sendChat()` |
| `Shift + Enter`（chat 输入框中） | 换行 | 默认行为 |

## 十、视觉元素

### 42. 音频波形
- **描述**：音频轨 clip 内显示随机生成的波形可视化
- **实现**：`drawWave(canvas)` — Canvas 2D 绘制多层正弦波 + 随机扰动，seed 基于 clip ID
- **状态**：已完成

### 43. 帧刻度线
- **描述**：视频 clip 内显示微弱的帧分割刻度
- **实现**：`makeFrameTicks(dur)` — 根据 clip 宽度生成 `.clip-frame` 列表
- **状态**：已完成

### 44. 轨道颜色标识
- **描述**：不同轨道类型的 clip 有不同灰度色调 + 轨道头左侧色条
- **实现**：
  - 视频轨：近白 (`cv1`) + 深色条
  - 音频轨：浅灰 (`cau`) + 浅灰条
  - 歌词轨：淡橙 (`cly`) + 橙色条（仅 mv-editor.html）

### 45. 卡片式面板
- **描述**：素材面板、预览区、时间轴区域均使用圆角卡片 + 微阴影，脱离全屏无边框风格
- **实现**：`border-radius: var(--r-lg)` + `box-shadow: var(--sh-sm)`，面板间 4px gap
- **状态**：已完成

### 46. 文件版本
- **mv-editor.html**（完整版）：多视频轨 + 歌词轨 + 新增歌词按钮 + 新增轨道按钮，传统全屏编辑器布局
- **mv-editor-simple.html**（精简版）：AI 对话面板（360px）+ 素材面板 + 预览区 + 单视频轨 + 单音频轨 + 撤销/重做，左中右三栏对话式布局
