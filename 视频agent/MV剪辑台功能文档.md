# MV 剪辑台 — 功能文档

> 基于 `mv-editor.html`（完整版：多轨道 + 歌词）和 `mv-editor-simple.html`（精简版：单视频轨+单音频轨）原型，记录所有交互功能。

---

## 一、顶部工具栏

### 1. 返回
- **描述**：返回上一页（浏览器历史）
- **触发**：点击左上角 `←` 图标按钮
- **实现**：`history.back()`
- **状态**：已完成

### 2. 导出成片
- **描述**：导出最终 MV 视频文件
- **触发**：点击右上角橙色「导出成片」按钮
- **实现**：`exportProject()` — 当前显示 toast 提示"导出功能开发中…"
- **状态**：占位（TODO）

## 二、素材面板（左侧 30%）

### 3. 素材分组切换
- **描述**：按 All / A / B / C 四组切换素材集
- **触发**：点击顶部 `All` `A` `B` `C` 标签
- **实现**：`switchPanel(btn, panel)` — 更新 `S.panel`，重新渲染素材网格
- **状态**：已完成

### 4. 素材类型切换
- **描述**：按 视频 / 图片 / 音频 筛选当前分组素材
- **触发**：点击 `视频` `图片` `音频` 标签
- **实现**：`switchType(btn, type)` — 更新 `S.type`，重新渲染素材网格
- **状态**：已完成

### 5. 导入素材
- **描述**：从电脑本地选择文件导入素材库，自动识别类型（video/image/audio），添加到当前分组
- **触发**：点击素材网格末尾的「导入素材」卡片（虚线边框）
- **实现**：`importFile()` — 创建 `<input type="file" multiple>`，遍历文件推入 `MEDIA[S.panel][type]`，去重后重新渲染
- **状态**：已完成

### 6. 素材拖拽到轨道
- **描述**：将素材从素材库拖入任意轨道，插入模式（非覆盖），自动按时间排序
- **触发**：拖拽素材卡片 → 释放到轨道行
- **实现**：HTML5 Drag-and-Drop API
  - `dragstart`：序列化 `{name, dur}` 到 `dataTransfer`
  - `dragover`：绘制橙色插入指示线 `drawDropIndicator()`
  - `drop`：计算落点时间 → 找到插入位置 → `splice` 插入 → `renderAllClips()`
- **状态**：已完成

## 三、视频预览区（右上）

### 7. 播放 / 暂停
- **描述**：控制播放状态，播放时按钮切换为暂停图标
- **触发**：点击播放按钮 或 视频画面 hover 播放覆盖层
- **实现**：`togglePlay()` — `setInterval` 以 100ms 步进更新 `S.t`，到达 `S.dur` 后自动归零
- **状态**：已完成

### 8. 进度条跳转
- **描述**：点击进度条任意位置，播放头跳转到对应时间
- **触发**：点击进度条区域
- **实现**：`seekFromBar(e)` — 计算点击位置百分比 → `S.t = p * S.dur` → 同步更新播放头 / 进度条 / 时间码 / 预览
- **状态**：已完成

### 9. 音量控制
- **描述**：弹出音量滑块，初始值 50%，范围 0-100%
- **触发**：点击音量 `🔊` 图标按钮
- **实现**：`openVol(e)` / `closeVol()` / `setVol(v)` — Popover 定位在按钮上方，range slider 驱动 `S.volume`
- **状态**：已完成

### 10. 倍速控制
- **描述**：弹出倍速选择器，可选 0.8x / 1x / 1.2x / 2x，初始 1x
- **触发**：点击 `1×` 按钮
- **实现**：`openSpd(e)` / `closeSpd()` / `setSpeed(spd)` — Radio 列表，选中项高亮橙色，更新 `S.speed` 影响播放速度
- **状态**：已完成

### 11. 全屏
- **描述**：进入/退出浏览器全屏
- **触发**：点击全屏图标按钮
- **实现**：`goFull()` — Fullscreen API：`document.documentElement.requestFullscreen()` / `document.exitFullscreen()`
- **状态**：已完成

### 12. 视频预览三级降级
- **描述**：根据素材生成状态，预览区显示不同内容
- **触发**：自动（点击选中 clip 时 / 播放过程中实时更新）
- **实现**：`updatePreview(label, opts, time)`
  - **Level 1 — 视频已生成** (`videoReady: true`)：显示渐变背景模拟视频画面 + 右上角「视频」badge
  - **Level 2 — 分镜图** (`videoReady: false, imageSeed: number`)：全屏渐变模拟分镜图 + 右上角「分镜图」badge
  - **Level 3 — 文字描述** (`videoReady: false, 无 imageSeed`)：居中白色 Libre Baskerville 文字，显示音乐结构标签 + 画面描述 + 底部"视频生成中"提示 + 右上角「描述」badge
- **状态**：已完成

### 13. 视频画面 hover 播放覆盖层
- **描述**：鼠标悬停在视频画面上时，显示半透明播放按钮覆盖层
- **触发**：hover 视频画面
- **实现**：CSS `:hover` 控制 `.video-hover-play` 的 opacity，点击调用 `togglePlay()`
- **状态**：已完成

### 14. 时间码显示
- **描述**：显示当前播放时间 / 总时长，如 `01:23 / 03:45`
- **触发**：自动更新
- **实现**：`updateTime()` — `fmtT(S.t)` 格式化，每 100ms 刷新
- **状态**：已完成

## 四、时间轴工具栏（底部上方）

### 15. 删除
- **描述**：删除当前选中的 clip
- **触发**：点击「删除」按钮 或 键盘 `Delete / Backspace`
- **实现**：`deleteSelected()` — 遍历 `CLIPS` 找到对应数据 splice 移除 → DOM remove → 清除 `S.sel`
- **状态**：已完成

### 16. 分割
- **描述**：在播放头位置将视频轨中跨越播放头的 clip 一分为二
- **触发**：点击「分割」按钮 或 键盘 `S`
- **实现**：`splitAtPlayhead()` — 遍历视频轨 clip，找到包含 `S.t` 的 clip，拆分为两个新 clip
- **状态**：已完成

### 17. 新增歌词字幕
- **描述**：AI 驱动，点击后立即显示歌词轨和 loading 状态，3 秒后填充歌词内容
- **触发**：点击「＋ 新增歌词字幕」按钮
- **实现**：`generateLyrics()`
  1. 显示歌词轨 header 和 track row
  2. 在轨内显示旋转 loading + "AI 正在分析歌词内容…"
  3. `setTimeout(3000ms)` 后写入 demo 歌词数据 → `renderAllClips()`
- **状态**：已完成（模拟 LLM 调用）

## 五、时间轴区域

### 18. 时间尺点击跳转
- **描述**：点击时间尺任意位置，播放头跳转到对应时间
- **触发**：点击时间尺（ruler）
- **实现**：`seekRuler(e)` — 计算相对于 ruler 的 x 坐标 + scrollLeft → `S.t = x / S.zoom`
- **状态**：已完成

### 19. 新增轨道
- **描述**：动态添加一个新的视频轨，自动编号（视频轨 3、4...），附带可拖入的 track row
- **触发**：点击轨道头底部的「＋ 轨道」按钮
- **实现**：`addTrack()` — 创建 `.tr-head` header + `.tr-row` row → 插入 DOM → 注册 drop target → `CLIPS[trackId] = []`
- **状态**：已完成

### 20. 时间轴缩放
- **描述**：Ctrl/Cmd + 滚轮缩放时间轴，clip 和时间尺同步缩放
- **触发**：Ctrl/Cmd + 鼠标滚轮
- **实现**：`setupScroll()` — `wheel` 事件监听 `e.ctrlKey || e.metaKey`，`S.zoom` 范围 4-80，缩放后重绘 ruler + clips + playhead
- **状态**：已完成

## 六、Clip 交互

### 23. Clip 选中
- **描述**：点击 clip 选中，显示橙色边框
- **触发**：鼠标点击 clip（非 handle 区域）
- **实现**：`onClipDown()` → `selectClip(el)` — 清除其他选中 → 添加 `.sel` class → 更新视频预览
- **状态**：已完成

### 24. Clip 拖动
- **描述**：拖拽 clip 在轨道内移动，松开后按新位置排序并紧密排列
- **触发**：鼠标按下 clip 并拖动
- **实现**：
  - 拖动中：`onDragMove()` type `move` — 计算 `dx` → 更新 `left`
  - 松开后：计算新起始时间（0.5s 吸附）→ 排序 → 紧靠（每个 clip 起始 = 前一个 clip 结束）
- **状态**：已完成

### 25. 初始自动紧靠
- **描述**：打开页面时所有轨道的 clip 自动紧密排列，无间隙
- **触发**：页面加载 `init()`
- **实现**：遍历所有轨道，按 start 排序后依次重算起始位置
- **状态**：已完成

### 25. Clip 缩放（调整时长）
- **描述**：拖拽 clip 左右两端的 resize handle，调整 clip 的起始时间和时长
- **触发**：鼠标按下 clip 左侧/右侧 handle 并拖动
- **实现**：`onDragMove()` type `resize` — 左侧 handle：同时改 `left` 和 `width`；右侧 handle：只改 `width`，最小宽度 20px
- **状态**：已完成

### 26. 磁性吸附（已替换）
- **描述**：已改为拖拽结束后的插入排序 + 紧密排列，不再使用运行时磁性吸附
- **状态**：已由 #25 替代

### 27. 点击空白取消选中
- **描述**：点击时间轴空白区域取消当前 clip 选中
- **触发**：点击时间轴非 clip 区域
- **实现**：全局 `click` 监听 — 如果 `!e.target.closest('.clip')` 且在 `.tl-scroll` 内，清除 `.sel` 和 `S.sel`
- **状态**：已完成

## 七、播放系统

### 29. 播放头实时更新
- **描述**：播放时橙色播放头随时间移动
- **触发**：播放中每 100ms
- **实现**：`updatePH()` — `playhead.style.left = S.t * S.zoom + 'px'`，使用 `will-change: left` 优化
- **状态**：已完成

### 30. 播放头自动滚动跟随
- **描述**：播放头移出可视区域时，时间轴自动滚动跟随
- **触发**：播放中每 100ms
- **实现**：`togglePlay()` 内 — 当 `phPx > scrollLeft + clientWidth - 80` 时设置 `scrollLeft = phPx - clientWidth / 2`
- **状态**：已完成

### 31. 倍速影响播放
- **描述**：选择的倍速影响播放步进速度
- **触发**：`setSpeed(spd)` 后播放时自动生效
- **实现**：`S.t += 0.1 * S.speed` — 倍速乘子直接作用于时间步进
- **状态**：已完成

### 32. 播放循环
- **描述**：播放到结尾后自动归零重新开始
- **触发**：`S.t >= S.dur` 时自动
- **实现**：`togglePlay()` 内 — 归零 `S.t = 0`，停止 timer，重置按钮图标
- **状态**：已完成

## 八、键盘快捷键

| 快捷键 | 功能 | 实现 |
|--------|------|------|
| `Space` | 播放 / 暂停 | `e.code === 'Space'` → `togglePlay()` |
| `Delete / Backspace` | 删除选中 clip | `e.code === 'Delete/Backspace'` → `S.sel.remove()` |
| `S` | 在播放头处分割 | `e.code === 'KeyS'` → `splitAtPlayhead()` |
| `Ctrl/Cmd + 滚轮` | 时间轴缩放 | `wheel` 事件 → 更新 `S.zoom` |

> `Shift + 拖拽` 临时关闭吸附 — 已随节拍网格移除

## 九、视觉元素

### 33. 音频波形
- **描述**：音频轨 clip 内显示随机生成的波形可视化
- **实现**：`drawWave(canvas)` — Canvas 2D 绘制多层正弦波 + 随机扰动，seed 基于 clip ID
- **状态**：已完成

### 34. 帧刻度线
- **描述**：视频 clip 内显示微弱的帧分割刻度
- **实现**：`makeFrameTicks(dur)` — 根据 clip 宽度生成 `.clip-frame` 列表
- **状态**：已完成

### 33. 轨道颜色标识
- **描述**：不同轨道类型的 clip 有不同灰度色调 + 轨道头左侧色条
- **实现**：
  - 视频轨：近白 (`cv1`) + 深色条
  - 音频轨：浅灰 (`cau`) + 浅灰条
  - 歌词轨：淡橙 (`cly`) + 橙色条（仅 mv-editor.html）

### 34. 文件版本
- **mv-editor.html**（完整版）：多视频轨 + 歌词轨 + 新增歌词按钮 + 新增轨道按钮
- **mv-editor-simple.html**（精简版）：单视频轨 + 单音频轨，无歌词相关功能，无新增轨道按钮
