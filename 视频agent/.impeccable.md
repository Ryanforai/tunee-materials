## Design Context

### Users
音乐创作者 / AI 音乐人 / MV 兴趣小白。在创作环境中使用，需要沉浸感。用户希望从灵感 → 方案 → 成片的完整创作流程中获得流畅、愉悦的体验。

### Brand Personality
创意、专业、沉浸感。像一个懂音乐的创作伙伴，不是冷冰冰的工具。

### Aesthetic Direction
- **主题**：浅色（参考 Claude 配色，明亮的创作工作台）
- **字体**：避免 Inter/DM Sans/IBM Plex 等 reflex 字体。选用有特色的无衬线 + 显示字体组合
- **色彩**：暖色调中性背景 + 纯白卡片 + 微妙边框，橙色（oklch 50 色相）强调色
- **参考氛围**：Claude Artifact 的清晰 + Notion 的结构化 + Linear 的克制
- **避免**：霓虹渐变、玻璃拟态、过度圆角卡片、纯色纯黑纯白

### Design Principles
1. **清晰优先**：浅色主题，信息层级一目了然
2. **专业但不冰冷**：温暖的色调和人性化的交互反馈
3. **信息层级清晰**：对话 → 画布 → 工具，逐层递进
4. **动效有意义**：只在状态变化时使用动画，不为装饰而装饰
