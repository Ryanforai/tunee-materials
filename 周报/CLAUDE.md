# 周报

存放每周工作周报，按周建文件。

## 文件规范

- 命名格式：`YYYY-MM-DD~YYYY-MM-DD.md`（周一至周日）
- 每周新建一个文件，不要覆盖旧的
- 内容按框架填写：本周概要 → 核心成果 → 问题与应对 → 下周重点 → 需要的支持
- 写法见 [`周报经验.md`](周报经验.md)
