# tunee-materials

Tunee MV Studio AI 资产库。包含 MV 分镜脚本提示词、视频生成 Skill、Claude Skill 合集和测试素材。

---

## 目录

| 目录 | 内容 |
|------|------|
| [`creative_analysis/`](creative_analysis/) | MV 分镜脚本生成与修改的 System Prompt 集合（生成类 + 修改类） |
| [`Tunee Skill库/`](Tunee%20Skill库/) | Claude Code Skill 包（cinematic-prompt-generator、free-music-generator） |
| [`视频skill/`](视频skill/) | MV 分镜生成 Skill 完整版（单节点 + 双节点，支持 lip-sync / story / visions） |
| [`视频测试/`](视频测试/) | 提示词研究文件、视觉参考、音频样本、需求文档 |

---
